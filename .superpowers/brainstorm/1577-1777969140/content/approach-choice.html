方案对比

自动布局有几种实现方式，各有优缺点

A

@dagrejs/dagre + React Flow

⭐ 推荐 — React Flow 生态的标准布局方案，轻量、成熟、效果好

B

ELK.js

更强大的图形布局引擎，适合复杂拓扑结构，但配置较繁琐

C

自定义简单布局

根据拓扑顺序计算位置，代码简单但效果粗糙

推荐方案 A 的优势

  • React Flow 官方推荐的布局方案
  • 安装量大的成熟库
  • 自动处理节点间距和层级
  • 代码改动最小
  • 支持水平和垂直两种方向

注意

  • 需要安装新依赖 @dagrejs/dagre
  • 每次加载/添加节点时触发重新布局
