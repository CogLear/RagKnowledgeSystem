### 1.2 会议室预约

**预约方式**：

| 预约渠道 | 操作步骤 |
|---------|---------|
| OA系统 | oa.company.com → 会议室预约 |
| 企业微信 | 工作台 → 会议室预约 |
| 移动端 | 企业微信小程序 → 会议室预约 |

**预约流程**：

1. 登录会议室预约系统
2. 选择需要的日期和时间
3. 选择会议室
4. 填写会议主题和参会人数
5. 确认预约
6. 系统自动发送确认邮件

**预约规则**：

- 提前预约：最多提前2周
- 最短预约：30分钟
- 最长预约：4小时
- 取消预约：提前15分钟可取消

### 1.3 会议室使用

**签到**：

- 进入会议室前在系统签到
- 15分钟内未签到，会议室自动释放
- 签到后系统延长预留时间

**使用注意**：

| 注意事项 | 说明 |
|---------|-----|
| 准时开始 | 预约时间开始后15分钟内签到 |
| 按时结束 | 预约时间结束前5分钟提醒 |
| 设备使用 | 按设备操作规范使用 |
| 卫生清洁 | 会议结束后带走垃圾 |
| 电源关闭 | 离开时关闭投影、空调 |

**异常处理**：

| 情况 | 处理方式 |
|-----|---------|
| 会议室被占用 | 联系行政部协调 |
| 设备故障 | 联系IT部门 |
| 需要额外设备 | 提前2小时联系行政部 |

## 四、访客接待

### 4.1 访客预约

**预约方式**：

- 邮件预约：来访者发送邮件至联系员工
- 电话预约：来访者电话联系确认
- 企业微信预约：发送访客邀请链接

**预约信息**：

| 信息项 | 说明 |
|-------|------|
| 访客姓名 | 全名 |
| 访客公司 | 公司全称 |
| 访客人数 | 预计人数 |
| 到访时间 | 日期和时间 |
| 被访人 | 我方联系人 |
| 来访目的 | 会议/参观/面试等 |

**预约确认**：

- 被访人确认后回复访客
- 发送访客邀请函
- 包含公司地址、交通指引、入门方式

### 4.2 访客到访流程

**到访当天**：

```
访客到达 → 前台登记 → 核实身份 → 发放访客卡 → 联系被访人 → 访客进入
```

**前台登记**：

1. 访客出示身份证或其他证件
2. 填写访客登记表
3. 确认被访人信息
4. 发放访客卡和临时工牌

**访客卡使用**：

- 访客卡 = 临时门禁权限
- 有效期：当天有效
- 权限范围：预约访问的区域

**会见安排**：

1. 前台联系被访人
2. 被访人到前台迎接
3. 带领访客进入会议室/办公区
4. 会见结束后送访客离开

### 4.3 访客停车

**停车场位置**：

- 地下停车场B2、B3层
- 访客车位在B3层

**停车费用**：

| 类型 | 费用 | 说明 |
|-----|------|-----|
| 访客停车 | 前2小时免费 | 超时10元/小时 |
| 面试停车 | 免费 | 需提前登记车牌号 |

**预约登记**：

- 访客来访前，被访人需在系统登记访客车牌号
- 前台核实后放行

### 4.4 面试候选人接待

**面试接待流程**：

1. HR提前发送面试邀请（含地址、交通）
2. 候选人到达前台
3. 前台电话联系HR或面试官
4. HR/面试官到前台迎接
5. 带领候选人前往会议室
6. 面试结束后送候选人离开

**等候安排**：

- 候选人等候区：3层休息区
- 提供茶水
- 等候时间不超过15分钟

**面试会议室**：

- 小会议室（4人）或第二会议室（10人）
- 提前15分钟预约

## 五、办公环境

### 5.1 办公室守则

**基本要求**：

- 着装得体，符合岗位要求
- 保持工位整洁
- 午餐请在休息区或食堂
- 物品借用需登记
- 爱护公共设施

**禁止行为**：

- 办公区吸烟（吸烟区在楼外）
- 带宠物进入办公区
- 在办公区烹饪
- 存放易燃易爆物品
- 占用公共区域堆放私人物品

### 5.2 空调使用

**温度设置**：

| 季节 | 温度设置 | 说明 |
|-----|---------|-----|
| 夏季 | 24-26℃ | 节能模式 |
| 冬季 | 20-22℃ | 保暖模式 |

**使用规则**：

- 下班后自动关闭
- 窗户开着时关闭空调
- 如有不适联系物业
- 禁止自行调整温度设置

**报修处理**：

- 空调故障联系物业
- 物业电话：010-12345680
- 紧急情况联系行政部

### 5.3 饮水服务

**饮水机位置**：

- 3层茶水间：饮水机1台
- 4层茶水间：饮水机1台
- 5层茶水间：饮水机1台

**使用注意**：

- 冷热水供应
- 热水温度较高，请注意安全
- 禁止洗漱或其他非饮用用途

**桶装水更换**：

- 联系物业更换
- 备用桶装水存放在茶水间角落
- 物业电话：010-12345680

### 5.4 绿植维护

**绿植配置**：

- 公司统一购置绿植
- 每周一次专业养护
- 绿植位置已规划，禁止随意移动

**枯萎处理**：

- 如发现绿植枯萎
- 联系行政部更换
- 禁止自行丢弃

### 5.5 清洁服务

**日常清洁**：

- 每天下班后清洁
- 工位垃圾统一清理
- 茶水间、卫生间每日消毒

**深度清洁**：

- 每周五下午进行深度清洁
- 地毯吸尘
- 玻璃擦拭

**垃圾处理**：

| 垃圾类型 | 处理方式 | 位置 |
|---------|---------|-----|
| 普通垃圾 | 垃圾桶 | 各楼层茶水间 |
| 可回收物 | 蓝色垃圾桶 | 各楼层茶水间 |
| 纸质文件 | 碎纸机/文件回收箱 | 打印区 |
| 电子垃圾 | 联系IT部门 | - |

## 六、物资管理

### 6.1 办公用品领用

**领用方式**：

| 用品类型 | 领用方式 | 数量限制 |
|---------|---------|---------|
| 笔、本子、便利贴 | 前台领取 | 每人每周5件 |
| 文件夹、回形针 | 前台领取 | 按需 |
| 大件用品 | IT工单申请 | 需审批 |

**领用流程**：

1. 前往3层前台
2. 告知需要的用品
3. 前台登记发放
4. 签字确认

### 6.2 物资借用

**借用范围**：

- 投影仪
- 摄像机
- 音响设备
- 打印机
- 测温仪
- 工具箱

**借用流程**：

1. 联系行政部预约
2. 填写借用登记表
3. 领取物资并检查
4. 按时归还
5. 行政部验收

**损坏处理**：

- 正常使用损坏：公司承担
- 人为损坏：责任人承担维修费
- 丢失：按原价赔偿
