# 安全与合规

本文档详细介绍信息安全、物理安全、合规管理、反贿赂反腐败等安全合规相关内容。

---

## 一、信息安全

### 1.1 信息安全目标

**核心目标**：

- 保护公司商业秘密和客户数据
- 确保信息系统安全稳定运行
- 遵守法律法规和行业规范
- 维护公司声誉和品牌形象

### 1.2 密码管理

**密码安全策略**：

| 系统 | 密码长度 | 必须包含 | 更换周期 |
|-----|---------|---------|---------|
| 企业邮箱 | 8位以上 | 大小写字母、数字 | 90天 |
| OA系统 | 8位以上 | 大小写字母、数字 | 90天 |
| 财务系统 | 12位以上 | 大小写字母、数字、特殊字符 | 90天 |
| VPN | 12位以上 | 大小写字母、数字、特殊字符 | 90天 |

**密码设置规范**：

```
✅ 正确示例：
- Abc12345
- Qwer!@#$

❌ 错误示例：
- 12345678
- password
- qwerty
- 姓名+生日
```

**密码安全行为**：

| ✅ 应该做 | ❌ 禁止做 |
|---------|---------|
| 使用复杂密码 | 使用简单密码 |
| 定期更换密码 | 多个系统用同一密码 |
| 密码不写在纸上 | 将密码告诉他人 |
| 使用密码管理器 | 在邮件/聊天中发送密码 |

### 1.3 账号安全

**账号使用规范**：

| 规范 | 说明 |
|-----|------|
| 禁止共享 | 账号密码不得共享给他人 |
| 禁止借用 | 不得使用他人账号 |
| 及时报告 | 账号异常立即报告IT |
| 定期检查 | 定期检查账号登录记录 |

**账号异常处理流程**：

```
发现异常 → 立即冻结账号（联系IT） → 修改密码 → 检查操作记录 → 解除冻结
```

### 1.4 文件安全

**文件分类分级**：

| 级别 | 定义 | 标识 | 处理方式 |
|-----|------|-----|---------|
| 机密 | 公司核心机密 | 红色标识 | 限制访问，加密存储 |
| 秘密 | 重要业务信息 | 蓝色标识 | 部门内共享，审批后外发 |
| 内部 | 内部使用 | 绿色标识 | 公司员工可访问 |
| 公开 | 对外发布 | 无标识 | 无限制 |

**文件保护措施**：

| 措施 | 说明 |
|-----|------|
| 加密存储 | 敏感文件必须加密 |
| 权限控制 | 按级别设置访问权限 |
| 水印保护 | 重要文件添加水印 |
| 审计日志 | 记录文件访问和操作 |

**文件外发规范**：

- 通过公司邮件系统外发
- 大文件使用加密压缩
- 注明保密级别和用途
- 跟踪文件使用情况

### 1.5 网络安全

**安全上网规范**：

| ✅ 应该做 | ❌ 禁止做 |
|---------|---------|
| 访问正规网站 | 访问非法网站 |
| 下载正版软件 | 下载盗版/破解软件 |
| 使用公司网络 | 使用P2P下载 |
| 及时更新系统 | 忽视系统更新 |

**网络异常处理**：

| 问题 | 处理方式 |
|-----|---------|
| 疑似中毒 | 断开网络，联系IT |
| 钓鱼邮件 | 不点击，报告IT |
| 勒索软件 | 立即断电，联系IT |
| 黑客攻击 | 断开网络，保护证据 |

### 1.6 数据安全

**数据分类**：

| 类型 | 示例 | 保护要求 |
|-----|------|---------|
| 客户数据 | 联系方式、订单信息 | 最高保护，禁止泄露 |
| 财务数据 | 收入、成本、预算 | 高保护，限制访问 |
| 员工数据 | 个人信息、薪资 | 高保护，HR专用 |
| 产品数据 | 代码、设计文档 | 中保护，内部共享 |
| 公开数据 | 新闻稿、宣传资料 | 正常保护 |

**数据备份**：

| 类型 | 频率 | 保存时间 |
|-----|------|---------|
| 每日备份 | 每天 | 保留30天 |
| 每周备份 | 每周 | 保留90天 |
| 每月备份 | 每月 | 保留1年 |
| 每年备份 | 每年 | 永久保存 |

## 二、物理安全

### 2.1 门禁管理

**门禁权限**：

| 区域 | 权限级别 | 可访问人员 |
|-----|---------|----------|
| 办公区 | 普通 | 全体员工 |
| 服务器机房 | 最高 | IT管理员 |
| 财务室 | 高 | 财务部+特批 |
| 档案室 | 高 | 行政部+特批 |
| 会议室 | 普通 | 全体员工 |

**门禁卡管理**：

| 规范 | 说明 |
|-----|------|
| 专人专用 | 门禁卡仅限本人使用 |
| 不得外借 | 禁止将门禁卡借给他人 |
| 遗失报告 | 遗失立即报告IT部门 |
| 离职归还 | 离职时交还行政部 |

**访客门禁**：

- 访客需由员工迎接
- 前台登记后发放临时门禁卡
- 临时卡仅限当天有效
- 离开时归还前台

### 2.2 物品出门管理

**出门条制度**：

| 物品类型 | 要求 | 说明 |
|---------|-----|-----|
| 笔记本等设备 | 必须开具出门条 | 经部门负责人签字 |
| 大件物品 | 必须开具出门条 | 经部门负责人签字 |
| 私人物品 | 无需出门条 | 保安抽查 |
| 文件资料 | 根据密级 | 机密文件禁止带出 |

**出门流程**：

```
领取出门条 → 填写物品信息 → 部门负责人签字 → 保安核查 → 放行
```

### 2.3 消防安全

**消防设施**：

| 设施类型 | 位置 | 检查周期 |
|---------|-----|---------|
| 灭火器 | 各楼层走廊 | 每月检查 |
| 消防栓 | 各楼层走廊 | 每季检查 |
| 烟雾报警器 | 各区域天花 | 每月检查 |
| 喷淋系统 | 办公区天花 | 每季检查 |
| 应急照明 | 各通道 | 每月检查 |

**消防通道**：

| 规范 | 说明 |
|-----|------|
| 保持畅通 | 通道不得堆放物品 |
| 禁止锁闭 | 安全出口不得锁闭 |
| 清晰标识 | 疏散路线清晰可见 |
| 定期演练 | 每半年一次消防演练 |

**火灾应急**：

```
发现火情 → 按报警器 → 拨打119 → 疏散人员 → 初期灭火 → 等待救援
```

**疏散流程**：

1. 保持镇定，不要惊慌
2. 用湿毛巾捂住口鼻
3. 沿疏散标识撤离
4. 不乘坐电梯，走楼梯
5. 到达指定集合点
6. 清点人数报告

### 2.4 应急预案

**各类应急预案**：

| 场景 | 应对措施 | 联系方式 |
|-----|---------|---------|
| 火灾 | 按指示撤离，不乘电梯 | 119 |

| 停电 | 等待电力恢复，使用应急照明 | 物业 |
| 漏水 | 关闭水源，报告物业 | 物业 |

**紧急联络**：

| 类型 | 电话 |
|-----|------|
| 火警 | 119 |
| 急救 | 120 |
| 报警 | 110 |

## 三、合规管理

### 3.1 反贿赂反腐败

**基本原则**：

| 原则 | 说明 |
|-----|------|
| 禁止受贿 | 严禁收受客户、合作方任何形式的贿赂 |
| 禁止行贿 | 严禁向政府官员、合作方提供贿赂 |
| 礼品申报 | 礼品价值超过200元需上报 |
| 举报保护 | 举报违规行为受法律保护 |

**礼品管理**：

| 场景 | 礼品价值 | 处理方式 |
|-----|---------|---------|
| 我方送客户 | 不限 | 需提前申请，计入招待费 |
| 客户送我方 | 200元以下 | 可接受 |
| 客户送我方 | 200元以上 | 拒绝或上交公司 |
| 现金/购物卡 | 任何价值 | 绝对禁止，一概拒绝 |

**宴请娱乐**：

| 类型 | 标准 | 审批要求 |
|-----|-----|---------|
| 一般宴请 | 200元/人 | 部门经理审批 |
| 重要客户 | 400元/人 | 总监审批 |
| 政府机关 | 500元/人 | VP审批 |

### 3.2 利益冲突

**需要申报的情形**：

| 类型 | 申报时间 | 说明 |
|-----|---------|-----|
| 兼职 | 入职及变更时 | 与公司业务相关需申报 |
| 投资 | 入职及变更时 | 投资与公司有往来企业 |
| 亲属就业 | 入职及变更时 | 亲属在本公司或关联公司 |
| 业务往来 | 发生时 | 与本人有利益关系的业务 |

**回避制度**：

- 员工应主动回避涉及自身利益的业务
- 无法回避时需向上级报告
- 由上级决定是否需要调整分工

### 3.3 知识产权

**职务成果**：

| 类型 | 归属 | 说明 |
|-----|------|-----|
| 发明专利 | 公司 | 工作期间的发明归公司 |
| 软件代码 | 公司 | 工作期间编写的代码归公司 |
| 设计文档 | 公司 | 工作期间创作归公司 |
| 职务作品 | 公司 | 以工作为主的创作归公司 |

**知识产权保护**：

| 措施 | 说明 |
|-----|------|
| 保密协议 | 入职签署保密协议 |
| 竞业限制 | 离职签署竞业限制协议 |
| 版权声明 | 文档标注版权信息 |
| 专利申请 | 发明及时申请专利 |

### 3.4 商业秘密

**商业秘密范围**：

| 类型 | 示例 | 保护等级 |
|-----|------|---------|
| 技术信息 | 产品配方、工艺流程、源代码 | 机密 |
| 经营信息 | 客户名单、价格策略、营销计划 | 机密 |

**保密措施**：

## 五、违规处理

### 5.1 违规分级

**严重违规**（直接解除）：

| 违规类型 | 示例 |
|---------|-----|
| 泄露机密 | 向外部泄露商业机密 |
| 贪污受贿 | 收受贿赂或行贿 |
| 欺诈 | 伪造文件、数据造假 |
| 暴力行为 | 殴打他人 |
| 盗窃 | 偷窃公司或他人财物 |

**一般违规**（视情况处理）：

| 违规类型 | 处理方式 |
|---------|---------|
| 迟到早退 | 扣款处理 |
| 旷工 | 扣款+警告 |
| 损坏公物 | 照价赔偿 |
| 违反流程 | 警告+培训 |
| 信息安全事故 | 视严重程度处理 |

### 5.2 违规处理流程

```
发现违规 → 调查核实 → 听取申辩 → 做出决定 → 通知本人 → 执行处理
```

**申诉权利**：

- 员工对处理决定有异议
- 可在收到通知后7天内提出申诉
- 由上级或HR部门重新审议

## 六、常见问题

### 6.1 信息安全问题

**Q：收到钓鱼邮件怎么办？**
A：不要点击链接或附件，立即报告IT部门。

**Q：密码忘了怎么办？**
A：通过自助服务重置，或联系IT部门。

**Q：发现信息泄露怎么办？**
A：立即报告IT安全和HR部门。

**Q：可以使用个人云存储吗？**
A：不可以，只能使用公司指定的企业网盘。

### 6.2 物理安全问题

**Q：门禁卡丢了怎么办？**
A：立即联系IT部门禁用旧卡，申请新卡。

**Q：发现可疑人员怎么办？**
A：通知前台或保安，不要直接对峙。

**Q：物品出门需要什么手续？**
A：需开具出门条，经部门负责人签字。

**Q：消防演习是强制的吗？**
A：是的，每半年一次的消防演习必须参加。

### 6.3 合规问题

**Q：客户送的礼物可以收吗？**
A：200元以下的可以收，超过200元需拒绝或上交。

**Q：可以从事副业吗？**
A：与公司业务相关的副业需申报，公司同意后才可从事。

**Q：竞业限制期有多长？**
A：通常为离职后2年，具体看合同约定。

**Q：发现同事违规怎么办？**
A：可以向HR或通过匿名举报渠道报告。
