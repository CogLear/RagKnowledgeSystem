# 测试知识库 - 公司通用知识手册

> 本文档为 RAG 系统测试用知识库，涵盖公司日常运营的各个方面。

## 一、入职指南

### 1.1 入职流程总览
新员工入职流程分为以下阶段：

**第一阶段：入职前准备**
1. 收到录用通知后，HR会发送入职邀请邮件
2. 准备相关证件和材料
3. 预约指定医院的入职体检

**第二阶段：入职当天**
1. 9:00前到达公司前台报到
2. 提交入职材料给HR
3. 签订劳动合同和保密协议
4. 领取工牌、门禁卡
5. 加入企业微信和钉钉群

**第三阶段：入职后一周内**
1. 完成IT账号开通（企业邮箱、OA、门禁）
2. 参加新员工培训
3. 分配工位和办公设备
4. 与直属上级进行入职面谈
5. 了解部门架构和团队成员

### 1.2 入职体检
- **指定医院**：朝阳区建国路88号北京市第一医院
- **体检时间**：周一至周六 8:00-11:00（空腹）
- **费用**：200元，入职后凭发票报销
- **需要携带**：身份证原件、一寸照片2张
- **体检项目**：血常规、尿常规、肝功能、胸片、心电图
- **注意事项**：体检前三天清淡饮食，体检当天空腹

### 1.3 入职准备材料清单
| 序号 | 材料名称 | 份数 | 要求 | 用途说明 |
|-----|---------|-----|------|---------|
| 1 | 身份证 | 2份复印件 | 正反面复印 | 存档、社保办理 |
| 2 | 学历证书 | 1份复印件 | 本科及以上 | 学历验证、存档 |
| 3 | 离职证明 | 1份原件 | 加盖原单位公章 | 社保转移、劳动关系转移 |
| 4 | 社保转移单 | 1份原件 | 当地社保局开具 | 社保账户合并 |
| 5 | 照片 | 3张 | 白底一寸 | 工牌、胸卡、档案 |
| 6 | 体检报告 | 1份原件 | 指定医院出具 | 入职审核 |
| 7 | 银行卡 | 1张复印件 | 正面复印件 | 工资发放 |
| 8 | 居住证 | 1份复印件 | 非本地户籍需提供 | 居住登记 |

### 1.4 试用期说明
- **试用期时长**：3个月
- **试用期工资**：正式工资的80%
- **绩效考核**：试用期每月一次绩效评估
- **转正评估**：试用期最后一周进行转正答辩
- **转正标准**：绩效评分≥70分，无重大违规违纪
- **转正后**：工资恢复100%，五险一金基数调整，享有过节费和年终奖

### 1.5 劳动合同签订
- 合同类型：固定期限劳动合同
- 合同期限：3年（试用期3个月包含在内）
- 签订时间：入职后3个工作日内
- 合同份数：一式两份，员工和公司各执一份
- 签订流程：HR准备→员工确认→双方签字→公司盖章→员工领取

## 二、考勤与假期

### 2.1 工作时间
| 项目 | 时间 | 说明 |
|-----|------|-----|
| 标准上班时间 | 9:00 | 弹性范围9:00-9:30 |
| 标准下班时间 | 18:00 | 弹性范围17:30-18:30 |
| 午休时间 | 12:00-13:00 | 1小时 |
| 周末双休 | 周六、周日 | 国家法定节假日按政府规定 |
| 弹性工作 | 需申请 | 适用于特定岗位，需部门负责人审批 |

### 2.2 打卡考勤
- **打卡方式**：门禁卡+企业微信定位打卡
- **补卡申请**：每月最多3次，需说明原因并经上级审批
- **缺卡处理**：未打卡且无补卡申请视为旷工
- **外勤打卡**：出差或外出可通过企业微信定位打卡

### 2.3 请假类型与规定

**年假政策**：
| 工龄 | 年假天数 | 说明 |
|-----|---------|-----|
| 试用期 | 按转正天数折算 | 入职当年度 |
| 满1年 | 5天 | 工作满1年后的首个周年起 |
| 满3年 | 10天 | 第三个周年起 |
| 满5年 | 15天 | 第五个周年起 |
| 满10年 | 20天 | 第十个周年起 |

**年假使用规则**：
- 年假需在当年度内使用，不可跨年累计超过5天
- 请假需提前3天申请，紧急情况可先电话通知
- 年假最小请假单位为半天

**病假规定**：
| 病假天数 | 要求 | 工资发放 |
|---------|-----|---------|
| 3天以内 | 提交医院诊断证明 | 100% |
| 3-7天 | 提交医院证明+病历 | 80% |
| 7天以上 | 需提供完整病历及检查报告 | 60% |
| 超过30天 | 需经过公司医务室评估 | 停发基本工资 |

**事假规定**：
- 全年累计不超过10个工作日
- 扣除当日全额工资
- 需提前3天申请，紧急情况需邮件说明
- 事假期间不享受餐补和交通补贴

**婚假规定**：
- 法定婚假3天（晚婚增加7天）
- 需提前1周申请
- 提交结婚证复印件
- 工资全额发放

**产假规定**：
- 顺产：98天基础+30天奖励假
- 难产：增加15天
- 多胞胎：每多一个增加15天
- 流产：怀孕不满4个月15天，4个月以上42天
- 陪产假：10天（晚婚晚育增加3天）
- 产假期间工资按生育保险标准发放

**丧假规定**：
| 亲属关系 | 假期天数 | 说明 |
|---------|---------|-----|
| 父母、配偶、子女 | 5天 | 含路程 |
| 祖父母、外祖父母、岳父母 | 3天 | 含路程 |
| 兄弟姐妹 | 2天 | 含路程 |

### 2.4 加班管理

**加班认定**：
- 工作日加班：下班时间18:00后继续工作
- 周末加班：周六周日工作
- 节假日加班：国家法定节假日工作

**加班调休规则**：
| 加班类型 | 调休时长 | 使用期限 |
|---------|---------|---------|
| 工作日加班 | 1:1 | 季度内使用 |
| 周末加班 | 1:2 | 半年内使用 |
| 节假日加班 | 1:3 | 年底前使用 |

**加班申请流程**：
1. 提前在OA系统提交加班申请
2. 注明加班原因和预计时长
3. 经直属上级审批
4. 人事部备案
5. 加班后3天内确认实际加班时长

**加班管控**：
- 单日加班不超过4小时
- 月度加班时长不超过36小时
- 超过36小时需总监级别审批

### 2.5 迟到早退与旷工

**迟到处理**：
| 迟到时长 | 处理方式 |
|---------|---------|
| 15分钟以内 | 每月3次弹性，超过按实际统计 |
| 15-30分钟 | 扣罚50元/次 |
| 30-60分钟 | 扣罚当日半天工资 |
| 60分钟以上 | 扣罚当日全天工资 |

**早退处理**：参照迟到处理标准执行

**旷工处理**：
- 连续旷工1天：扣罚3天工资
- 连续旷工3天：视为严重违纪，解除劳动合同
- 旷工期间不享受餐补、交通补贴

## 三、薪酬福利

### 3.1 工资结构
```
税前月工资 = 基本工资 + 岗位工资 + 绩效奖金 + 各类补贴 + 加班费
```

| 工资组成部分 | 说明 | 占比（示例） |
|-------------|-----|-------------|
| 基本工资 | 岗位对应的基本薪酬 | 50% |
| 岗位工资 | 职级对应的岗位津贴 | 20% |
| 绩效奖金 | 根据绩效考核结果发放 | 0-30% |
| 餐补 | 每天30元，按月发放 | 约10% |
| 交通补贴 | 每月500元 | 固定 |
| 通讯补贴 | 每月200元 | 固定 |
| 其他补贴 | 高温补贴、取暖补贴等 | 按实际 |

### 3.2 工资发放
- **发放时间**：每月15日发放上月工资（如遇节假日提前至前一个工作日）
- **发放方式**：直接打入员工绑定银行卡
- **工资条**：每月通过邮件发送电子工资条，可在OA系统查看明细
- **年终奖**：根据公司年度业绩和个人绩效，通常在农历年前发放

### 3.3 五险一金
**缴纳基数**：按员工上年度月平均工资计算，上限为当地社会平均工资的3倍

| 险种 | 个人缴纳 | 公司缴纳 | 备注 |
|-----|---------|---------|-----|
| 养老保险 | 8% | 16% | 累计缴纳满15年可领取 |
| 医疗保险 | 2% | 10% | 有个人账户，可药店购药 |
| 失业保险 | 0.5% | 0.5% | 非自愿失业可领取 |
| 工伤保险 | 0% | 0.5% | 工伤认定后报销 |
| 生育保险 | 0% | 0.8% | 生产后报销相关费用 |
| 住房公积金 | 12% | 12% | 购房、租房可提取 |

**社保转移**：
1. 入职后由HR办理社保转入
2. 需要原单位提供社保转移单
3. 转移周期约1-2个月
4. 公积金账户转移需员工本人办理

### 3.4 其他福利

**节日福利**：
| 节日 | 过节费标准 | 发放时间 |

## 四、财务报销

### 4.1 报销流程
```
费用发生 → 取得发票 → 财务系统录入 → 上传附件 → 部门审批 → 财务复核 → 出纳打款
```

**详细流程**：
1. **费用发生**：员工因公产生费用
2. **取得发票**：必须为合法有效发票
3. **财务系统录入**：登录OA财务系统，填写报销单
4. **上传附件**：拍照或扫描上传发票
5. **部门审批**：部门负责人审核
6. **财务复核**：财务部审核发票真伪和报销标准
7. **出纳打款**：复核通过后3个工作日内打款

**报销时效**：
- 费用发生后30天内必须提交报销
- 超期需部门总监特批
- 跨年发票不予报销

### 4.2 差旅报销标准

**交通费用**：
| 出行方式 | 职级 | 标准 | 备注 |
|---------|-----|------|-----|
| 飞机 | 普通员工 | 经济舱 | 需提前申请 |
| 飞机 | 经理级 | 经济舱 | 优先折扣票 |
| 飞机 | 总监级 | 经济舱/商务舱 | 可申请商务舱 |
| 高铁 | 普通员工 | 二等座 | 优先高铁 |
| 高铁 | 经理级及以上 | 一等座 | 可申请商务座 |
| 火车 | 所有员工 | 硬卧/软卧 | 超过6小时可选软卧 |
| 出租车 | 所有员工 | 实报实销 | 需说明起始地点 |
| 租车 | 所有员工 | 200元/天上限 | 需提前审批 |
| 市内交通 | 所有员工 | 100元/天 | 包含地铁、公交、打车 |

**住宿标准**：
| 城市级别 | 普通员工 | 经理级 | 总监级 |
|---------|---------|--------|--------|
| 北上广深 | 350元/晚 | 500元/晚 | 800元/晚 |
| 省会城市 | 250元/晚 | 350元/晚 | 500元/晚 |
| 其他城市 | 200元/晚 | 300元/晚 | 400元/晚 |

**餐费标准**：
| 出差类型 | 标准 | 说明 |
|---------|-----|-----|
| 当日往返 | 100元 | 不需要住宿 |
| 1-3天出差 | 150元/天 | 包干使用 |
| 3天以上出差 | 120元/天 | 第三天起降低标准 |

**差旅补贴**：
- 交通补贴：50元/天（不含市内交通）
- 通讯补贴：10元/天

### 4.3 业务招待费用

**招待标准**：
| 招待对象 | 人均标准 | 陪同人数 | 说明 |
|---------|---------|---------|-----|
| 政府机关 | 500元/人 | 不超过3人 | 需提前申请 |
| 重要客户 | 400元/人 | 不超过5人 | 需提前申请 |
| 普通客户 | 200元/人 | 不超过3人 | 部门经理审批 |

**招待申请流程**：
1. 提前1天在OA系统提交招待申请
2. 注明招待对象、目的、人数
3. 经直属上级和财务审批
4. 超标准招待需总监级审批

### 4.4 办公费用报销

**办公用品**：
- 单次100元以内：部门自行采购，凭发票报销
- 单次100元以上：需通过公司采购系统下单

**快递费用**：
- 公司业务快递：月结，由公司统一支付
- 个人快递：自付

**打印复印**：
- 工作需要：部门统一结算
- 个人使用：从工资中扣减

### 4.5 发票要求与规范

**发票类型**：
| 发票类型 | 可报销 | 说明 |
|---------|-------|-----|
| 增值税专用发票 | ✅ | 最佳，发票联和抵扣联都要 |
| 增值税普通发票 | ✅ | 抬头必须正确 |
| 电子发票 | ✅ | 需打印后提交，PDF也可 |
| 手撕发票 | ✅ | 需刮开密码 |
| 定额发票 | ⚠️ | 部分情况可接受 |
| 收据 | ❌ | 不可作为报销凭证 |
| 白条 | ❌ | 不可作为报销凭证 |

**发票内容要求**：
- 购买方名称：公司全称
- 纳税人识别号：91110108MA00XXXXXX
- 开票内容：与实际消费一致
- 开票金额：与实际支付一致
- 开票日期：30天内有效
- 开票方信息：清晰完整

**发票问题处理**：
| 问题类型 | 处理方式 |
|---------|---------|
| 发票抬头错误 | 联系开票方作废重开 |
| 发票金额错误 | 联系开票方作废重开 |
| 发票丢失 | 复印件+对方税务证明+情况说明 |
| 超期发票 | 不予报销 |

## 五、IT支持

### 5.1 账号与权限管理

**企业邮箱**：
- 格式：姓名全拼@company.com（如：zhangsan@company.com）
- 开通时间：入职后1个工作日内
- 初始密码：身份证后6位（首次登录必须修改）
- 容量：无限容量
- 别名：可设置1-2个别名

**系统账号**：
| 系统名称 | 开通时间 | 用途 |
|---------|---------|-----|
| OA系统 | 入职当天 | 考勤、审批、报销 |
| 财务系统 | 入职后3天 | 费用报销 |
| 项目管理系统 | 入职后3天 | 项目协作 |
| 代码仓库(Git) | 入职后3天 | 代码管理（技术岗） |
| VPN | 申请后开通 | 远程办公 |

**账号申请流程**：
1. 在IT工单系统提交申请
2. 选择需要的系统和权限
3. 直属上级审批
4. IT部门处理（1-3个工作日）

### 5.2 设备管理

**办公设备配置**：
| 岗位类型 | 电脑配置 | 显示器 | 其他 |
|---------|---------|--------|-----|
| 行政/职能 | 笔记本 i5/8G/256G | 23寸×1 | - |
| 销售/市场 | 笔记本 i5/8G/512G | 23寸×1 | - |
| 研发/技术 | 台式机 i7/16G/512G 或 MacBook Pro | 27寸×2 | 机械键盘 |
| 设计 | iMac 27寸 或 MacBook Pro | - | 专业显示器 |

**设备申请**：
- 正常申请：IT工单系统，3个工作日内处理
- 紧急申请：发送邮件至IT部门，注明原因
- 更换周期：电脑4年一换，显示器6年一换

**设备维修**：
- 硬件故障：提交IT工单，2小时内响应
- 软件问题：提交IT工单或电话支持
- 紧急故障：拨打IT热线 010-12345679

**设备归还**：
- 离职前1天联系IT部门
- 归还所有公司资产（电脑、充电器、门禁卡等）
- 如有损坏或丢失需照价赔偿

### 5.3 网络与安全

**办公网络**：
| 网络名称 | 适用人群 | 带宽 | 说明 |
|---------|---------|-----|-----|
| Corp-WiFi | 全体员工 | 100M | 需要域账号连接 |
| Corp-5G | 全体员工 | 200M | 5G频段，信号更强 |
| Corp-Guest | 访客 | 20M | 无需密码，限时使用 |
| Corp-Secure | 特殊岗位 | 100M | 需要额外申请 |

**VPN使用**：
- 适用场景：远程办公、分公司互联
- 申请方式：IT工单系统申请
- 使用方法：安装VPN客户端，输入账号密码
- 连接时长：单次最长8小时

**安全规范**：
- 密码要求：8位以上，含大小写字母和数字
- 密码更新：每90天强制更新

- 离开工位必须锁屏（Win+L或Command+L）
- 敏感文件需加密存储
- 禁止使用未经授权的云存储

### 5.4 常见问题处理

**无法连接WiFi**：
1. 确认是否在办公区域内
2. 检查WiFi开关是否打开
3. 尝试忘记网络后重新连接
4. 确认域账号密码正确（注意大小写）
5. 尝试重启电脑或手机
6. 如仍无法解决，联系IT部门

**无法访问外网**：
1. 检查浏览器代理设置（应为空或自动检测）
2. 确认是否连接公司VPN
3. 尝试访问百度等常用网站
4. 联系IT部门处理

**邮箱问题**：
| 问题 | 解决方法 |
|-----|---------|
| 无法登录邮箱 | 检查密码，尝试找回密码 |
| 收不到邮件 | 检查垃圾邮件箱，确认邮箱地址正确 |
| 邮件发送失败 | 检查收件人地址，确认附件大小（不超过50M） |
| 邮箱满 | 清理邮件或申请扩容 |

**打印问题**：
1. 确认打印机电源和网络连接
2. 在设备列表中选择正确打印机
3. 打印驱动是否安装正确
4. 联系IT部门远程支持

**会议投屏问题**：
1. 检查投屏设备电源
2. 选择正确的信号源（HDMI/VGA/无线）
3. 笔记本按Win+P选择投屏模式
4. 无线投屏需连接Corp-投屏网络

Q：可以申请远程办公吗？
A：特定岗位可申请，需部门负责人和HR审批，每周不超过2天。

Q：忘记打卡怎么办？
