## 一、账号与权限管理

### 1.1 企业邮箱

**邮箱地址格式**：

```
姓名全拼@company.com
```

**示例**：

- 张三 → zhangsan@company.com
- 李四明 → lisiming@company.com
- 王芳 → wangfang@company.com

**特殊情况**：

- 重名员工：在姓名后加数字，如 zhangsan2@company.com
- 英文名：可申请英文邮箱别名

**邮箱功能**：

| 功能 | 说明 |
|-----|------|
| 容量 | 无限容量 |
| 别名 | 最多2个别名 |
| 转发 | 可设置自动转发 |
| 群组 | 可创建邮件群组 |

**初始密码**：

- 密码为身份证后6位（X用0代替）
- 首次登录必须修改密码
- 密码要求：8位以上，含大小写字母和数字

**邮箱开通时间**：

- 入职当天1个工作日内开通
- 开通后HR会发送账号信息到个人邮箱

### 1.2 系统账号

**常用系统列表**：

| 系统名称 | 开通时间 | 用途 | 申请方式 |
|---------|---------|-----|---------|
| OA系统 | 入职当天 | 审批、考勤、报销 | 自动开通 |
| 财务系统 | 入职后3天 | 费用报销、付款 | IT工单申请 |
| 项目管理系统 | 入职后3天 | 项目协作、任务管理 | IT工单申请 |
| 代码仓库(Git) | 入职后3天 | 代码管理（技术岗） | 直属上级申请 |
| VPN | 申请后开通 | 远程办公 | IT工单申请 |
| 会议室系统 | 入职当天 | 会议室预约 | 自动开通 |

**账号申请流程**：

1. 登录IT工单系统（it.company.com）
2. 选择"账号权限申请"
3. 选择需要的系统和权限
4. 填写申请理由
5. 直属上级审批
6. IT部门处理（1-3个工作日）

### 1.3 权限管理

**权限类型**：

| 权限级别 | 说明 | 适用人群 |
|---------|-----|---------|
| 普通权限 | 基础功能使用 | 普通员工 |
| 高级权限 | 高级功能、管理功能 | 经理、主管 |
| 管理员权限 | 系统配置、用户管理 | IT管理员 |

**权限申请**：

- 在IT工单系统提交申请
- 说明需要的具体权限
- 直属上级和IT共同审批

**权限回收**：

- 离职当天自动回收所有权限
- 岗位变动权限随之调整
- 权限使用异常可申请禁用

## 三、网络配置

### 3.1 办公网络

**网络类型**：

| 网络名称 | 适用人群 | 带宽 | 连接方式 | 说明 |
|---------|---------|-----|---------|-----|
| Corp-WiFi | 全体员工 | 100M | 域账号连接 | 需要域账号和密码 |
| Corp-5G | 全体员工 | 200M | 5G频段 | 信号更强，推荐使用 |
| Corp-Guest | 访客 | 20M | 短信验证码 | 无需密码，限时4小时 |
| Corp-Secure | 特殊岗位 | 100M | 需要额外申请 | 金融、敏感数据岗位 |

**WiFi连接步骤**：

1. 打开设备WiFi
2. 选择 "Corp-WiFi" 或 "Corp-5G"
3. 输入域账号和密码
4. 点击连接

**域账号说明**：

- 域账号 = 企业邮箱前缀
- 密码 = 邮箱密码
- 注意大小写

### 3.2 VPN使用

**VPN适用场景**：

- 远程办公（在家、出差）
- 分公司互联
- 访问公司内网系统

**VPN申请**：

1. IT工单系统提交VPN申请
2. 说明使用场景
3. 直属上级审批
4. IT开通（1个工作日内）

**VPN使用方法**：

1. 下载并安装VPN客户端
2. 配置VPN服务器地址
3. 输入账号密码
4. 连接成功

**VPN使用时长**：

- 单次最长连接8小时
- 超过后自动断开
- 需要可重新连接

### 3.3 网络问题排查

**无法连接WiFi**：

1. 确认设备WiFi开关打开
2. 确认在办公区域内
3. 尝试忘记网络后重新连接
4. 确认域账号密码正确（注意大小写）
5. 重启电脑或手机
6. 如仍无法解决，联系IT部门

**无法访问外网**：

1. 检查浏览器代理设置（应为空或自动检测）
2. 确认是否连接VPN
3. 尝试访问 baidu.com
4. 重启浏览器
5. 联系IT部门

**网络经常掉线**：

1. 检查是否在WiFi信号弱区
2. 尝试切换到Corp-5G
3. 联系IT部门检查网络设备

## 四、安全规范

### 4.1 密码管理

**密码要求**：

| 系统 | 密码长度 | 必须包含 | 其他要求 |
|-----|---------|---------|---------|
| 企业邮箱 | 8位以上 | 大小写字母、数字 | 禁止使用弱密码 |
| OA系统 | 8位以上 | 大小写字母、数字 | 禁止使用弱密码 |
| 财务系统 | 12位以上 | 大小写字母、数字、特殊字符 | 每90天强制更换 |
| VPN | 12位以上 | 大小写字母、数字、特殊字符 | 每90天强制更换 |

**弱密码禁止列表**：

- 123456、password、qwerty
- 企业名称、生日、电话号码
- 用户名相同或倒序

**密码更换**：

- 财务系统、VPN：每90天强制更换
- 企业邮箱：首次登录强制更换
- 更换密码不能与最近5次相同

**密码找回**：

- 企业邮箱：https://password.company.com 自助找回
- OA系统：联系IT部门重置
- 其他系统：IT工单申请重置

### 4.2 账户安全

**禁止行为**：

- 共享账号密码给他人
- 使用他人账号登录
- 在非公司设备登录公司账号
- 将账号密码写在便签或文档中

**账号异常处理**：

1. 发现账号异常立即联系IT
2. IT会临时禁用账号
3. 修改密码后重新启用
4. 检查是否有异常操作

### 4.3 文件安全

**敏感文件处理**：

- 必须加密存储
- 使用公司指定的加密工具
- 加密密码不得与账号密码相同

**文件外发**：

- 通过公司邮件外发
- 大文件使用公司网盘
- 禁止使用个人邮箱发送

**云存储使用**：

| 类型 | 是否允许 | 说明 |
|-----|---------|-----|
| 公司企业网盘 | ✅ 允许 | 推荐使用 |
| 钉钉云盘 | ✅ 允许 | 推荐使用 |
| 百度网盘 | ❌ 禁止 | 存在安全风险 |
| Dropbox | ❌ 禁止 | 存在安全风险 |
| OneDrive | ❌ 禁止 | 存在安全风险 |

### 4.4 网络行为规范

**禁止行为**：

- 访问非法网站
- 下载非法软件
- 使用P2P下载（迅雷、BT等）
- 观看不适当内容
- 进行大流量下载（影响网络）

**带宽使用**：

- 工作优先：邮件、OA系统、项目管理
- 合理使用：视频会议、在线文档
- 限制使用：大文件下载、在线视频

## 五、常见问题处理

### 5.1 邮箱问题

**无法登录邮箱**：

| 原因 | 解决方法 |
|-----|---------|
| 密码错误 | 找回密码或联系IT |
| 账号被锁 | 联系IT解锁 |
| 网络问题 | 检查网络连接 |

**收不到邮件**：

1. 检查垃圾邮件箱
2. 确认邮箱地址正确
3. 确认邮件未被拦截
4. 联系IT检查

**邮件发送失败**：

1. 检查收件人地址
2. 确认附件大小（不超过50M）
3. 检查网络连接
4. 重新发送

**邮箱满了**：

- 清理不需要的邮件
- 删除垃圾箱
- 联系IT扩容

### 5.2 打印问题

**无法连接打印机**：

1. 确认打印机电源和网络连接
2. 在设备列表中选择正确打印机
3. 确认打印驱动安装正确
4. 重启打印服务

**打印质量差**：

1. 更换墨盒/硒鼓
2. 清洁打印机
3. 联系IT更换耗材

**常见打印机位置**：

| 位置 | 打印机名称 | 说明 |
|-----|----------|-----|
| 3层茶水间 | HP-3F-01 | 供3层使用 |
| 4层茶水间 | HP-4F-01 | 供4层使用 |
| 5层茶水间 | HP-5F-01 | 供5层使用 |
| 会议室301 | HP-301 | 仅供会议室 |

### 5.3 会议设备问题

**投影仪无法使用**：

1. 检查投影仪电源
2. 选择正确信号源（HDMI1/HDMI2/VGA）
3. 笔记本按 Win+P 选择投影模式
4. 更换数据线尝试

**投屏失败**：

1. 确认连接 Corp-投屏 WiFi
2. 重启投屏设备
3. 确认设备支持投屏协议
4. 联系IT支持

**视频会议问题**：

| 问题 | 解决方法 |
|-----|---------|
| 听不到声音 | 检查扬声器和音量设置 |
| 麦克风无声 | 检查麦克风权限和设置 |
| 视频不显示 | 检查摄像头权限和驱动 |
| 会议掉线 | 检查网络连接，尝试重拨 |

### 5.4 软件安装

**可以安装的软件**：

| 软件类型 | 示例 | 安装方式 |
|---------|-----|---------|
| 办公软件 | Office、钉钉、企业微信 | IT工单申请 |
| 开发工具 | VS Code、Git | IT工单申请 |
| 浏览器 | Chrome、Firefox | IT工单申请 |

**禁止安装的软件**：

- 盗版软件
- 游戏软件
- 破解工具
- P2P下载软件
- 与工作无关的软件

**软件安装申请流程**：

1. IT工单系统提交申请
2. 说明软件名称和用途
3. IT评估安全性
4. IT协助安装或提供安装包
