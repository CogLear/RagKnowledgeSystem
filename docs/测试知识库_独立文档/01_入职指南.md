# 入职指南

本文档详细介绍新员工入职流程、材料准备、试用期说明等入职相关知识。

---

## 一、入职流程总览

新员工入职流程分为以下阶段，每个阶段都有明确的时间和任务要求。

### 1.1 入职前准备阶段

**收到录用通知后**，HR会在3个工作日内发送入职邀请邮件，邮件包含：

- 入职日期和时间安排
- 入职当天联系人及联系方式
- 公司地址和交通指引
- 需要准备的材料清单
- 体检医院信息

**入职前1周**：

1. 确认邮件回复（点击邮件中的确认链接）
2. 预约指定医院的入职体检
3. 准备所有证件原件和复印件
4. 准备一寸白底照片3张
5. 办理原单位社保转移手续

**入职前3天**：

1. HR会发送欢迎邮件，包含工号、企业微信邀请码
2. 开通企业邮箱预注册（可提前设置邮箱别名）
3. 选好心仪的工位区域（3楼层有可选）

### 1.2 入职当天流程

**9:00 到达公司**

- 地点：公司前台（1层大堂）
- 联系人：前台接待 或 HR小李（工号10001）
- 联系电话：010-12345671

**9:00-9:30 前台登记**

1. 出示身份证原件
2. 签署访客登记表
3. 领取访客卡和临时工牌
4. 拍照制作正式工牌（费用5元，从当月工资扣除）

**9:30-10:30 HR办理入职**

1. 验证所有证件原件与复印件
2. 提交照片和体检报告
3. 填写员工信息登记表
4. 签署劳动合同（3年期，试用期3个月）
5. 签署保密协议和竞业限制协议（如适用）
6. 提交银行卡复印件（用于工资发放）
7. 社保和公积金办理说明

**10:30-11:00 领取物资**

到6层行政部领取：

| 物资 | 数量 | 说明 |
|-----|-----|-----|
| 工牌 | 1个 | 含门禁功能 |
| 门禁卡 | 1张 | 开通权限 |
| 企业邮箱说明 | 1份 | 含初始密码 |
| 员工手册 | 1本 | 电子版也会发送到邮箱 |

**11:00-12:00 IT开通账号**

到4层IT部门：

1. 企业邮箱账号确认
2. OA系统账号开通
3. 门禁卡权限设置
4. 电脑和显示器领取
5. 网络连接和域账号设置

**12:00-13:00 午休时间**

公司食堂位于B1层，用餐免费。入职当天可领取餐卡（卡内无余额，需先充值）。

**13:00-14:00 工位安排**

1. 前往3层工位区
2. 安装调试电脑
3. 连接显示器和外设
4. 测试网络和邮箱

**14:00-15:00 部门报到**

1. 前往所属部门
2. 与直属上级见面
3. 了解团队成员
4. 领取办公用品（笔、本子、文件夹等）

**15:00-16:30 新员工培训**

由HR组织，内容包括：

- 公司介绍和历史
- 企业文化和价值观
- 组织架构和部门职能
- 规章制度（考勤、报销、行为规范）

**16:30-17:00 培训测试**

- 培训内容测试（20道选择题）
- 及格线：80分
- 不及格者需第二天补考

**17:00-18:00 自由安排**

- 熟悉办公环境
- 认识相邻工位同事
- 联系HR小李（有任何问题）

## 二、入职体检

### 2.1 体检医院信息

**指定医院**：北京市朝阳区建国路88号第一医院（体检中心）

| 项目 | 详情 |
|-----|------|
| 地址 | 朝阳区建国路88号，地铁大望路站B口出，步行600米 |
| 电话 | 010-87654321 |
| 营业时间 | 周一至周六 8:00-11:00（采血截止10:30） |
| 费用 | 200元（入职后凭发票报销） |

### 2.2 预约方式

1. 电话预约（推荐）：拨打医院体检中心电话
2. 网上预约：关注医院微信公众号预约
3. 现场预约：可能需要排队，不推荐

预约时说明是"公司入职体检"，医院会安排相应套餐。

### 2.3 体检前准备

**饮食要求**：

- 体检前3天清淡饮食，避免高脂肪、高蛋白食物
- 体检前1天晚餐19:00前进食，之后禁食
- 体检当天早晨禁食、禁水（降压药可少量水服用）

**携带材料**：

- 身份证原件
- 一寸白底照片2张
- 体检费用（200元，可支付宝/微信）

**穿着要求**：

- 穿宽松易脱的衣物（避免连衣裙、连体裤）
- 不要戴金属首饰
- 不要佩戴隐形眼镜

### 2.4 体检项目

| 序号 | 项目 | 检查内容 | 目的 |
|-----|-----|---------|-----|
| 1 | 采血 | 血常规、肝功能、肾功能、血糖、血脂 | 检查血液系统、代谢功能 |
| 2 | 尿常规 | 尿蛋白、尿糖、尿潜血 | 检查泌尿系统、代谢异常 |
| 3 | 胸片 | 胸部X光 | 检查肺部和胸廓 |
| 4 | 心电图 | 静态心电图 | 检查心脏电生理 |
| 5 | 腹部B超 | 肝胆胰脾肾 | 检查腹部脏器 |
| 6 | 内科 | 血压、心肺听诊、腹部触诊 | 检查基本健康状况 |
| 7 | 外科 | 身高体重、甲状腺、淋巴结 | 检查体表和外科情况 |
| 8 | 五官科 | 视力、听力、鼻喉 | 检查感官功能 |

### 2.5 体检结果

- 体检结果通常3个工作日后可查
- 医院会电话通知本人
- 如有异常项目，医院会建议复查
- 凭体检发票原件到HR处报销

### 2.6 特殊情况

**女性员工**：

- 怀孕或疑似怀孕需提前告知，避免做胸片
- 生理期可暂缓做尿常规

**疾病患者**：

- 如有慢性病（如高血压、糖尿病），体检当日正常服药
- 携带近期检查报告供医生参考

## 三、入职材料清单

### 3.1 必须提交的材料

| 序号 | 材料名称 | 份数 | 规格要求 | 用途说明 |
|-----|---------|-----|---------|---------|
| 1 | 身份证 | 2份复印件 | 正反面复印在同一页 | 存档、社保办理 |
| 2 | 学历证书 | 1份复印件 | 本科及以上需提供 | 学历验证、存档 |
| 3 | 离职证明 | 1份原件 | 加盖原单位公章 | 劳动关系转移、社保转移 |
| 4 | 社保转移单 | 1份原件 | 当地社保局开具 | 社保账户合并 |
| 5 | 照片 | 3张 | 白底一寸 | 工牌、胸卡、档案 |
| 6 | 体检报告 | 1份原件 | 指定医院出具 | 入职审核 |
| 7 | 银行卡 | 1张复印件 | 正面复印 | 工资发放 |
| 8 | 居住证 | 1份复印件 | 非本地户籍需提供 | 居住登记 |

### 3.2 材料准备注意事项

**身份证**：

- 有效期必须在有效期内
- 如果过期需先更新
- 临时身份证也可，但需在有效期内

**学历证书**：

- 大专及以上学历需要提供
- 需要学信网可查的学历
- 如有遗失，需提供学历证明（学信网认证报告）

**离职证明**：

- 必须为原件，加盖原单位公章
- 如原单位已倒闭，需提供失业证明或社保停缴记录
- 实习经历不需要离职证明

**社保转移单**：

- 由原单位社保关系转移至社保局开具
- 如已在本地参保，可不提供（直接办理转移）

**银行卡**：

- 需为本人名下银行卡
- 建议使用中国银行、工商银行、农业银行等大银行
- 不支持信用卡

### 3.3 材料提交时间

- **入职当天**：必须提交所有材料
- **如有缺失**：需在入职后3个工作日内补齐
- **逾期未补**：HR会通知部门负责人，可能影响试用期内工资发放

## 四、试用期说明

### 4.1 试用期时长

根据劳动合同法规定：

| 合同期限 | 试用期时长 | 说明 |
|---------|-----------|-----|
| 不满3个月 | 不得超过15天 | 短期合同 |
| 3个月-1年 | 不得超过1个月 | 一年期合同 |
| 1年-3年 | 不得超过2个月 | 中期合同 |
| 3年以上 | 不得超过6个月 | 长期合同 |

**本公司规定**：

- 所有新员工试用期统一为3个月
- 劳动合同期限为3年
- 试用期包含在劳动合同期限内

### 4.2 试用期工资

**工资计算**：

- 试用期工资 = 转正后工资 × 80%
- 不低于当地最低工资标准

**工资发放日**：每月15日（发放上月整月工资）

**其他福利**：

| 福利项目 | 试用期 | 转正后 |
|---------|-------|-------|
| 餐补 | 30元/天 | 30元/天 |
| 交通补贴 | 500元/月 | 500元/月 |
| 通讯补贴 | 200元/月 | 200元/月 |
| 五险一金 | 正常缴纳 | 正常缴纳 |
| 年假 | 按转正天数折算 | 正常享受 |
| 过节费 | 正常发放 | 正常发放 |

### 4.3 试用期考核

**月度评估**：

- 每月5日前提交上月工作小结
- 直属上级进行绩效面谈
- 考核结果记入员工档案

**转正评估**：

- 试用期最后一周进行转正答辩
- 答辩委员会：直属上级、HR、部门同事
- 评估维度：工作能力、工作态度、团队协作

### 4.4 转正标准

**基本条件**：

1. 绩效考核平均分≥70分
2. 培训测试通过（80分及以上）
3. 无重大违规违纪行为
4. 所有入职材料齐全

**优先条件**：

1. 提前完成工作任务
2. 提出建设性改进建议
3. 获得客户或同事好评

### 4.5 转正流程

**试用期最后10天**：

1. 员工提交转正申请表
2. 直属上级填写评估意见
3. HR审核材料完整性

**试用期最后5天**：

1. 转正答辩（30分钟）
2. 答辩内容包括：工作成果展示、问题解答、未来规划
3. 答辩委员会评分

**试用期结束当天**：

1. 通过转正：收到转正通知邮件，工资恢复100%
2. 未通过转正：延长试用期（不超过1个月）或解除劳动合同

## 五、劳动合同签订

### 5.1 合同类型

**本公司使用的劳动合同类型**：

- 固定期限劳动合同（3年期）
- 试用期3个月包含在合同期内

### 5.2 合同内容

劳动合同主要包含以下条款：

| 条款类别 | 主要内容 |
|---------|---------|
| 合同双方 | 公司名称、地址；员工姓名、住址、身份证号 |
| 合同期限 | 起始日期、结束日期、试用期 |
| 工作内容 | 岗位名称、工作地点、职责描述 |
| 工作时间 | 标准工时制、不定时工时制、综合工时制 |
| 休息休假 | 工作时间、休息日、法定节假日、年假 |
| 劳动报酬 | 工资标准、支付时间、加班费计算 |
| 社会保险 | 五险一金缴纳 |
| 劳动保护 | 工作条件、职业危害防护 |
| 合同变更 | 合同变更的条件和程序 |
| 合同解除 | 双方解除合同的条件和程序 |
| 违约责任 | 违约金、赔偿条款 |
| 竞业限制 | 离职后的竞业限制（适用于特定岗位） |

### 5.3 签订流程

**入职当天**：

1. HR准备劳动合同（一式两份）
2. 员工仔细阅读合同条款
3. 员工在合同上签字（一式两份）
4. HR代表公司盖章
5. 公司留存一份，员工留存一份

**注意事项**：

- 合同文本公司会提前发给员工查看
- 员工有权要求HR解释合同条款
- 签订前确认工资、岗位、工作地点等关键信息
- 合同签订后员工可拍照留存

### 5.4 合同变更

**可以变更的情形**：

1. 员工岗位调整（升职/平调）
2. 工作地点变更
3. 工作内容调整
4. 工资调整

**变更流程**：

1. 双方协商一致
2. 签订劳动合同变更协议
3. 变更协议一式两份

## 六、常见问题

### 6.1 入职前问题

**Q：可以推迟入职吗？**
A：如有特殊情况，可在收到offer后3天内联系HR申请延期，延期最长不超过2周。

**Q：入职当天有事来不了怎么办？**
A：提前联系HR小李（010-12345671），说明情况并重新预约时间。

**Q：材料不全可以先入职吗？**
A：身份证和离职证明是必须的，其他材料可在3天内补齐。

**Q：体检可以用医保卡吗？**
A：入职体检属于就业体检，不在医保范围内，需要自费。

### 6.2 入职当天问题

**Q：迟到会有影响吗？**
A：入职第一天弹性较大，但建议提前10分钟到达。如有突发情况联系HR。

**Q：可以带家人参观吗？**
A：不可以，访客需要提前预约，且不能进入办公区。

**Q：中午可以外出吃饭吗？**
A：可以，但需要携带工牌，便于门禁出入。

### 6.3 试用期问题

**Q：试用期可以请假吗？**
A：可以，按正常流程在OA系统提交申请。

**Q：试用期可以参加公司培训吗？**
A：可以，所有员工培训均对试用期员工开放。

**Q：试用期被证明不符合要求怎么办？**
A：公司会提前通知，提供改进机会。如仍不达标，可延长试用期或解除合同。

**Q：试用期可以辞职吗？**
A：可以，提前3天书面通知即可。
