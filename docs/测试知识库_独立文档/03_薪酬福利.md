# 薪酬福利制度

本文档详细介绍公司薪酬结构、工资发放、五险一金、节日福利等内容。

---

## 一、薪酬结构

### 1.1 工资组成

公司实行结构化工资制，员工的月工资由以下部分组成：

```
税前月工资 = 基本工资 + 岗位工资 + 绩效奖金 + 各类补贴 + 加班费
```

**工资结构明细**：

| 组成部分 | 说明 | 占比范围 | 发放时间 |
|---------|-----|---------|---------|
| 基本工资 | 根据岗位和职级确定 | 40-50% | 每月15日 |
| 岗位工资 | 体现岗位职责和技能要求 | 15-25% | 每月15日 |
| 绩效奖金 | 根据绩效考核结果浮动 | 0-30% | 每月15日 |
| 餐补 | 按出勤天数发放 | 固定 | 每月15日 |
| 交通补贴 | 月度固定发放 | 固定 | 每月15日 |
| 通讯补贴 | 月度固定发放 | 固定 | 每月15日 |
| 加班费 | 根据加班时长计算 | 浮动 | 每月15日 |

### 1.2 常见岗位工资示例

**技术序列**：

| 职级 | 基本工资 | 岗位工资 | 绩效奖金（浮动） |
|-----|---------|---------|----------------|
| 初级工程师 | 8,000 | 2,000 | 0-2,000 |
| 中级工程师 | 12,000 | 3,000 | 0-4,000 |
| 高级工程师 | 18,000 | 5,000 | 0-6,000 |
| 资深工程师 | 25,000 | 7,000 | 0-8,000 |

**运营序列**：

| 职级 | 基本工资 | 岗位工资 | 绩效奖金（浮动） |
|-----|---------|---------|----------------|
| 运营专员 | 6,000 | 1,500 | 0-1,500 |
| 运营主管 | 10,000 | 3,000 | 0-3,000 |
| 运营经理 | 18,000 | 5,000 | 0-5,000 |

### 1.3 年终奖

**计算方式**：

```
年终奖 = 月基本工资 × 月数 × 绩效系数 × 公司业绩系数
```

| 绩效等级 | 绩效系数 | 说明 |
|---------|---------|-----|
| A | 2.0-3.0 | 卓越表现 |
| B | 1.2-1.5 | 优秀 |
| C | 1.0 | 达标 |
| D | 0.5 | 需改进 |
| E | 0 | 不达标 |

**公司业绩系数**：

- 根据公司年度经营情况确定
- 通常在0.8-1.2之间
- 亏损年份可能为0

**发放时间**：

- 每年农历年前（1月底-2月中旬）
- 分两批发放：第一批80%，第二批20%（6个月后发放）

## 二、工资发放

### 2.1 发放时间

| 项目 | 时间 | 说明 |
|-----|------|-----|
| 发放日 | 每月15日 | 如遇节假日提前至前一个工作日 |
| 发放内容 | 上月1日至月末工资 | 非当月工资 |
| 工资到账 | 发放日当天 | 具体时间取决于银行处理速度 |

**示例**：

- 3月15日发放：2月1日-2月29日的工资
- 1月15日发放：12月1日-12月31日的工资

### 2.2 发放方式

**发放渠道**：

- 直接打入员工绑定的银行卡
- 支持银行：中国银行、工商银行、农业银行、建设银行、交通银行、招商银行、民生银行

**发放记录**：

- 工资条通过邮件发送至企业邮箱
- 邮件主题：「工资条-2024年XX月」
- 附件为PDF格式，可打印

### 2.3 工资条查看

**查看方式**：

1. 登录企业邮箱，查看工资条邮件
2. 登录OA系统 → 个人中心 → 工资条查询
3. 财务系统（finance.company.com）查看明细

**工资条内容**：

| 项目 | 说明 |
|-----|------|
| 基本信息 | 姓名、工号、部门、岗位 |
| 应发项目 | 基本工资、岗位工资、绩效奖金、补贴等 |
| 扣款项目 | 个人社保、个人公积金、个税 |
| 实发工资 | 应发工资 - 扣款 = 到账金额 |

## 三、五险一金

### 3.1 缴纳规定

**缴纳基数**：

- 按员工上年度月平均工资计算
- 上限：当地社会平均工资的3倍
- 下限：当地最低工资标准

**缴纳比例**：

| 险种 | 个人缴纳 | 公司缴纳 | 合计 |
|-----|---------|---------|-----|
| 养老保险 | 8% | 16% | 24% |
| 医疗保险 | 2% | 10% | 12% |
| 失业保险 | 0.5% | 0.5% | 1% |
| 工伤保险 | 0% | 0.5% | 0.5% |
| 生育保险 | 0% | 0.8% | 0.8% |
| 住房公积金 | 12% | 12% | 24% |

**示例计算**：

假设员工月薪10000元（低于上限）：

| 项目 | 个人缴纳 | 公司缴纳 |
|-----|---------|---------|
| 养老保险 | 800元 | 1,600元 |
| 医疗保险 | 200元 | 1,000元 |
| 失业保险 | 50元 | 50元 |
| 工伤保险 | 0元 | 50元 |
| 生育保险 | 0元 | 80元 |
| 住房公积金 | 1,200元 | 1,200元 |
| **合计** | **2,250元** | **3,980元** |

### 3.2 社保转移

**入职转移**：

1. 入职后HR会通知办理社保转入
2. 员工需提供原单位社保转移单
3. HR协助办理转入手续
4. 转移周期：1-2个月

**离职转出**：

1. 离职当月社保由公司缴纳
2. 离职后社保账户封存
3. 新单位入职后直接续交
4. 如跨城市就业，需办理转移手续

### 3.3 公积金

**公积金账户**：

- 入职后由HR协助开通
- 账户余额可通过公积金官网查询
- 初始密码：身份证后6位

**公积金提取**：

| 提取条件 | 提取方式 | 说明 |
|---------|---------|-----|
| 购房 | 按月冲还贷/一次性提取 | 需提供购房合同 |
| 租房 | 每月1500元定额提取 | 需提供租房合同 |
| 还贷 | 按月冲还贷 | 公积金直接冲抵月供 |
| 离职 | 账户封存6个月后提取 | 需提供离职证明 |

## 七、工资个税

### 7.1 个税计算

**计算公式**：

```
应纳税所得额 = 月工资 - 5000 - 社保公积金 - 专项附加扣除
应纳税额 = 应纳税所得额 × 税率 - 速算扣除数
```

**专项附加扣除**：

| 项目 | 扣除标准 | 说明 |
|-----|---------|-----|
| 子女教育 | 1000元/月/子女 | 3岁以下婴幼儿 |
| 继续教育 | 400元/月 | 学历教育 |
| 大病医疗 | 80000元以内据实 | 医保后自付超15000 |
| 住房贷款 | 1000元/月 | 首套房 |
| 住房租金 | 800-1500元/月 | 城市规模 |
| 赡养老人 | 2000元/月 | 独生子女 |

### 7.2 税率表

| 级数 | 应纳税所得额 | 税率 | 速算扣除数 |
|-----|-------------|-----|----------|
| 1 | 0-36000 | 3% | 0 |
| 2 | 36000-144000 | 10% | 2520 |
| 3 | 144000-300000 | 20% | 16920 |
| 4 | 300000-420000 | 25% | 31920 |
| 5 | 420000-660000 | 30% | 52920 |
| 6 | 660000-960000 | 35% | 85920 |
| 7 | 960000以上 | 45% | 181920 |

## 八、常见问题

### 8.1 工资问题

**Q：工资发放日是节假日怎么办？**
A：提前至前一个工作日发放。

**Q：发现工资有误怎么办？**
A：3个工作日内联系HR或财务，核对后会在下月补发或扣除。

**Q：试用期工资怎么算？**
A：试用期工资为转正工资的80%，其他福利不变。

**Q：离职当月工资怎么算？**
A：按实际出勤天数计算，日工资 = 月工资 ÷ 21.75。

### 8.2 福利问题

**Q：体检可以换钱吗？**
A：不可以，体检福利只能使用，不能折算现金。

**Q：团建费可以累积吗？**
A：不可以，按月发放，不可累积，不可跨月使用。

**Q：商业保险门诊可以报销吗？**
A：可以，社保范围内的门诊90%报销，社保外80%报销。

**Q：过节费需要扣税吗？**
A：需要，过节费并入当月工资一起计税。

### 8.3 社保问题

**Q：社保可以断缴吗？**
A：不可以断缴，断缴会影响看病报销、买房、落户等。

**Q：生育保险怎么用？**
A：产检和分娩时使用社保卡，生育津贴由社保基金支付。

**Q：公积金可以异地使用吗？**
A：可以，需办理公积金转移手续，或在购房地公积金中心申请异地贷款。
