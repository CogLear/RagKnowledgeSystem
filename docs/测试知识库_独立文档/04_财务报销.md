# 财务报销制度

本文档详细介绍公司费用报销流程、差旅报销标准、发票要求等内容。

---

## 一、报销流程总览

### 1.1 报销流程步骤

```
费用发生 → 取得发票 → 财务系统录入 → 上传附件 → 部门审批 → 财务复核 → 出纳打款
```

**详细说明**：

| 步骤 | 操作人 | 操作内容 | 时效要求 |
|-----|-------|---------|---------|
| 1. 费用发生 | 员工 | 因公消费 | - |
| 2. 取得发票 | 员工 | 向商家索取发票 | 即时 |
| 3. 财务系统录入 | 员工 | OA财务系统填写报销单 | 30天内 |
| 4. 上传附件 | 员工 | 拍照/扫描上传发票 | 30天内 |
| 5. 部门审批 | 直属上级 | 审核费用合理性 | 2个工作日内 |
| 6. 财务复核 | 财务部 | 审核发票真伪和标准 | 3个工作日内 |
| 7. 出纳打款 | 财务部出纳 | 打入工资卡 | 3个工作日内 |

### 1.2 报销时效

**报销时限**：

- 费用发生后30天内必须提交报销
- 超期不予报销（特殊情况需总监特批）
- 跨年发票不予报销（12月的发票最晚次年1月15日前）

**审批时效**：

| 审批节点 | 时效要求 |
|---------|---------|
| 直属上级 | 2个工作日内审批 |
| 部门总监 | 3个工作日内审批 |
| 财务复核 | 3个工作日内完成 |
| 出纳打款 | 审批通过后3个工作日内 |

### 1.3 报销入口

**报销系统**：

- OA系统：oa.company.com → 财务报销
- 财务系统：finance.company.com
- 移动端：企业微信 → 工作台 → 报销

## 二、差旅报销

### 2.1 差旅申请

**出差前**：

1. 在OA系统提交"出差申请"
2. 填写出差目的地、事由、时间、同行人
3. 预估费用（交通、住宿、餐饮、其他）
4. 直属上级审批
5. 财务备案

**出差中**：

1. 保留所有交通票据
2. 住宿开具正规发票
3. 餐饮尽量取得发票（小额可事后补）

**出差后**：

1. 7天内完成报销
2. 填写"差旅报销单"
3. 上传所有票据

### 2.2 交通费用报销

**飞机票**：

| 职级 | 可报销舱位 | 购票要求 | 说明 |
|-----|----------|---------|-----|
| 普通员工 | 经济舱 | 提前7天申请 | 优先折扣票 |
| 经理级 | 经济舱 | 提前申请 | 可选择舒适舱位 |
| 总监级 | 经济舱/商务舱 | 可申请商务舱 | 需说明原因 |

**火车票**：

| 职级 | 可报销舱位 | 说明 |
|-----|----------|-----|
| 普通员工 | 二等座 | 超过6小时可申请一等座 |
| 经理级 | 一等座 | 可申请商务座 |
| 总监级 | 商务座 | - |

**其他交通**：

| 交通方式 | 报销标准 | 说明 |
|---------|---------|-----|
| 出租车 | 实报实销 | 需说明起始地点 |
| 租车 | 200元/天上限 | 需提前审批 |
| 市内交通 | 100元/天上限 | 含地铁、公交、打车 |
| 高铁动车 | 二等座/一等座 | 按职级 |

### 2.3 住宿费用报销

**住宿标准**：

| 城市级别 | 普通员工 | 经理级 | 总监级 |
|---------|---------|--------|--------|
| 北上广深 | 350元/晚 | 500元/晚 | 800元/晚 |
| 省会城市 | 250元/晚 | 350元/晚 | 500元/晚 |
| 其他城市 | 200元/晚 | 300元/晚 | 400元/晚 |

**超标处理**：

- 超出标准部分自理
- 如有特殊情况需提前申请
- 超标50%以上需总监审批

**住宿要求**：

- 必须为正规酒店
- 出差地有协议酒店可享受折扣
- 入住天数 = 实际入住晚数

### 2.4 餐饮费用报销

**餐费标准**：

| 出差类型 | 餐费标准 | 说明 |
|---------|---------|-----|
| 当日往返 | 100元/天 | 不需要住宿 |
| 1-3天出差 | 150元/天 | 包干使用 |
| 3天以上出差 | 120元/天 | 第三天起降低标准 |

**报销方式**：

- 按日包干，无需提供发票
- 午餐超过30元/餐需提供发票

### 2.5 差旅补贴

| 补贴类型 | 标准 | 说明 |
|---------|-----|-----|
| 交通补贴 | 50元/天 | 不含市内交通 |
| 通讯补贴 | 10元/天 | 出差期间 |
| 其他杂费 | 30元/天 | 邮寄、寄存等 |

## 三、业务招待费

### 3.1 招待费用标准

**招待分类**：

| 招待对象 | 人均标准 | 陪同人数限制 | 说明 |
|---------|---------|------------|-----|
| 政府机关 | 500元/人 | 不超过3人 | 需提前申请 |
| 重要客户 | 400元/人 | 不超过5人 | 需提前申请 |
| 普通客户 | 200元/人 | 不超过3人 | 部门经理审批 |

**招待申请流程**：

1. 提前1天在OA系统提交招待申请
2. 填写招待对象、目的、人数、预计金额
3. 直属上级审批
4. 超标准招待需总监级审批

### 3.2 招待费报销

**报销材料**：

- 招待申请表（系统打印）
- 发票原件
- 消费明细（水单）
- 参与人员名单（如超过2人）

**不予报销**：

- 无招待申请的招待费用
- 超标准招待未说明原因
- 虚假招待

## 四、日常费用报销

### 4.1 办公费用

**办公用品**：

| 金额范围 | 报销方式 | 说明 |
|---------|---------|-----|
| 100元以内 | 自行采购，凭票报销 | 月累计不超过500元 |
| 100元以上 | 公司采购系统下单 | 部门统一采购 |

**快递费用**：

| 类型 | 报销方式 | 说明 |
|-----|---------|-----|
| 公司业务快递 | 月结，公司统一支付 | 联系前台登记 |
| 个人快递 | 自付 | 公司不报销 |

**打印复印**：

- 工作需要：公司统一结算
- 个人使用：从工资中扣减（0.2元/张）

### 4.2 通讯费用

**报销标准**：

| 岗位类型 | 通讯补贴 | 说明 |
|---------|---------|-----|
| 普通员工 | 200元/月 | 含在工资中发放 |
| 经理级 | 300元/月 | 含在工资中发放 |
| 总监级 | 500元/月 | 含在工资中发放 |

**特殊通讯费**：

- 因公产生的大额通讯费（如国际长途）
- 需提供通话记录和发票
- 单独报销，不计入通讯补贴

### 4.3 交通费用

**因公外出交通**：

- 出租车：实报实销，需说明起始地点
- 停车费：实报实销，需提供发票
- 加油费：仅限公司车辆，私车公用需申请

**交通补贴**：

- 月度交通补贴500元（随工资发放）
- 包含日常上下班和外出办事

## 五、发票要求

### 5.1 可报销的发票类型

| 发票类型 | 可否报销 | 说明 |
|---------|---------|-----|
| 增值税专用发票 | ✅ 可以 | 最佳，发票联和抵扣联都要 |
| 增值税普通发票 | ✅ 可以 | 抬头必须正确 |
| 电子发票 | ✅ 可以 | 需打印后提交，PDF也可 |
| 手撕发票 | ✅ 可以 | 需刮开密码 |
| 定额发票 | ⚠️ 部分 | 部分情况可接受 |
| 收据 | ❌ 不可以 | 不可作为报销凭证 |
| 白条 | ❌ 不可以 | 不可作为报销凭证 |

### 5.2 发票内容要求

**必须包含的信息**：

| 项目 | 要求 |
|-----|------|
| 购买方名称 | 公司全称 |
| 纳税人识别号 | 91110108MA00XXXXXX |
| 开票内容 | 与实际消费一致 |
| 开票金额 | 与实际支付一致 |
| 开票日期 | 30天内有效 |
| 开票方信息 | 清晰完整 |
| 发票章 | 清晰可见 |

**开票内容规范**：

| 消费类型 | 开票内容 | 备注 |
|---------|---------|-----|
| 交通 | "交通费"或具体交通类型 | 火车票/机票可直接报销 |
| 住宿 | "住宿费"或"住宿服务费" | 需注明入住天数 |
| 餐饮 | "餐饮费"或"餐费" | 招待费需开"招待费" |
| 办公用品 | "办公用品"或具体名称 | 商品明细需列明 |
| 快递 | "快递费" | 需注明寄送地点 |

### 5.3 发票问题处理

| 问题类型 | 处理方式 |
|---------|---------|
| 发票抬头错误 | 联系开票方作废重开 |
| 发票金额错误 | 联系开票方作废重开 |
| 发票丢失 | 复印件+对方税务证明+情况说明 |
| 超期发票 | 不予报销 |
| 发票抬头为个人 | 不予报销 |
| 假发票 | 不予报销，情节严重追责 |

### 5.4 电子发票报销

**电子发票要求**：

1. 需打印后提交（彩色或黑白均可）
2. 也可直接提交PDF文件
3. 发票内容需清晰可辨

**重复报销防范**：

- 电子发票需在发票背面注明"已报销"
- 同一张电子发票不可重复报销

## 六、借款管理

### 6.1 借款类型

**备用金借款**：

- 用于频繁的小额采购
- 额度：1000-5000元
- 周期：30天内冲账

**差旅借款**：

- 用于大额差旅预支
- 额度：根据实际需要
- 周期：出差回来7天内报销冲账

**项目借款**：

- 用于项目专项支出
- 额度：根据项目预算
- 周期：根据项目周期

### 6.2 借款流程

```
提交借款申请 → 直属上级审批 → 财务审批 → 出纳打款 → 按时冲账
```

**借款申请**：

1. OA系统提交借款申请
2. 填写借款金额、用途、预计还款时间
3. 上传相关支持文件

**还款/冲账**：

- 借款后按约定时间报销冲账
- 30天内未冲账将提醒
- 60天未冲账从工资中扣除

## 七、报销审核

### 7.1 审核要点

**合规性审核**：

- 费用是否在预算范围内
- 是否符合公司报销标准
- 是否按流程申请和审批

**真实性审核**：

- 发票是否真实有效
- 费用与业务是否相关
- 金额与事实是否一致

**规范性审核**：

- 发票内容是否规范
- 附件是否完整
- 签字是否齐全

### 7.2 不予报销的情况

| 情况 | 说明 |
|-----|------|
| 发票不合规 | 假发票、过期发票、抬头错误 |
| 未经审批 | 未按流程申请 |
| 超标未批 | 超出标准且无审批 |
| 与业务无关 | 个人消费、非因公支出 |
| 超报销时限 | 超过30天 |
| 重复报销 | 同一发票已报销 |

### 7.3 报销常见问题

**退单原因**：

1. 发票抬头或税号错误
2. 发票内容与报销类别不符
3. 缺少附件或明细
4. 金额与票据不一致
5. 超标准未提前审批

## 八、常见问题

### 8.1 报销时间问题

**Q：报销多久能到账？**
A：审批通过后3个工作日内打款。

**Q：发票丢了怎么办？**
A：需联系开票方补开发票，或提供其他支付凭证+情况说明。

**Q：超期发票能报销吗？**
A：不能，发票30天内有效，跨期不予报销。

### 8.2 报销标准问题

**Q：超标住宿可以报销吗？**
A：超标部分自理，特殊情况需提前申请并获得批准。

**Q：没有发票怎么报销？**
A：无法报销，必须提供合法有效发票。

**Q：电子发票可以报销吗？**
A：可以，需打印后提交或提交PDF。

### 8.3 报销流程问题

**Q：出差途中垫付的钱怎么办？**
A：保留票据，回来后按流程报销。

**Q：可以申请预支差旅费吗？**
A：可以，出差前申请借款，回来后报销冲账。

**Q：报销单填错了怎么办？**
A：在财务复核前可以撤回修改，重新提交。

### 8.4 特殊报销问题

**Q：招待客户的费用谁来买单？**
A：一般由招待方（我方）承担，保留发票报销。

**Q：加班餐费可以报销吗？**
A：可以，按30元/次报销，需说明加班原因。

**Q：客户礼品可以报销吗？**
A：可以计入招待费，需提前申请，标准参照招待费标准。
