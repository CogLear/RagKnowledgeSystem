# 绩效考核制度

本文档详细介绍绩效考核维度、评分标准、考核流程、结果应用等绩效考核相关内容。

---

## 一、绩效考核概述

### 1.1 考核目标

绩效考核旨在：

- 促进公司战略目标达成
- 客观评价员工工作表现
- 激励员工持续改进
- 为薪酬、晋升提供依据
- 发现和培养人才

### 1.2 考核周期

| 考核类型 | 时间 | 内容 |
|---------|-----|-----|
| 月度考核 | 每月5日前 | 提交上月工作小结 |
| 季度考核 | 每季度末月 | 正式绩效评估 |
| 年度考核 | 每年12月 | 全年绩效总评 |
| 项目考核 | 项目结束 | 项目绩效评估 |

### 1.3 考核原则

| 原则 | 说明 |
|-----|------|
| 客观公正 | 以事实和数据为依据 |
| 双向沟通 | 上级与下属充分沟通 |
| 持续改进 | 关注成长和发展 |
| 差异化 | 区分优秀和一般 |
| 及时反馈 | 考核结果及时反馈 |

## 二、考核维度

### 2.1 通用考核维度

| 维度 | 权重 | 说明 |
|-----|-----|-----|
| 业绩目标 | 50% | 完成工作任务的量和质 |
| 工作能力 | 30% | 专业技能、沟通协作等 |
| 工作态度 | 20% | 责任心、主动性等 |

### 2.2 各维度详解

**业绩目标维度**：

| 考核要素 | 具体指标 |
|---------|---------|
| 任务完成 | 按时按质完成工作任务 |
| 工作效率 | 单位时间产出 |
| 工作质量 | 结果的准确性和完整性 |
| 创新贡献 | 提出改进建议并实施 |

**工作能力维度**：

| 能力类别 | 考核要素 |
|---------|---------|
| 专业能力 | 技术水平、业务知识 |
| 沟通能力 | 表达清晰、倾听理解 |
| 协作能力 | 团队合作、跨部门配合 |
| 解决问题 | 分析问题、决策能力 |
| 学习能力 | 学习成长、知识分享 |

**工作态度维度**：

| 态度要素 | 考核内容 |
|---------|---------|
| 责任心 | 认真负责、主动承担 |
| 积极性 | 主动工作、乐于奉献 |
| 纪律性 | 遵守制度、服从安排 |
| 服务意识 | 内部客户满意度 |

### 2.3 岗位差异化权重

**技术序列**：

| 维度 | 权重 | 调整说明 |
|-----|-----|---------|
| 业绩目标 | 60% | 技术产出更易量化 |
| 工作能力 | 30% | 强调专业技术能力 |
| 工作态度 | 10% | 基本要求 |

**管理序列**：

| 维度 | 权重 | 调整说明 |
|-----|-----|---------|
| 业绩目标 | 40% | 团队成果为主 |
| 领导能力 | 30% | 团队管理和培养 |
| 工作态度 | 10% | 文化践行 |
| 战略能力 | 20% | 部门规划和决策 |

## 三、评分标准

### 3.1 绩效等级

| 等级 | 描述 | 比例限制 | 奖金系数 |
|-----|------|---------|---------|
| A | 卓越，超出预期 | ≤10% | 1.5 |
| B | 优秀，满足预期 | 20-30% | 1.2 |
| C | 良好，基本达标 | 40-50% | 1.0 |
| D | 需改进，部分不达标 | 10-15% | 0.8 |
| E | 不达标 | ≤5% | 0.5 |

### 3.2 评分细则

**业绩目标评分**：

| 等级 | 完成度 | 描述 |
|-----|-------|-----|
| A | 150%+ | 大幅超出预期 |
| B | 120-150% | 超预期完成 |
| C | 100-120% | 达成预期 |
| D | 80-100% | 基本达成，有差距 |
| E | <80% | 未达成预期 |

**能力评估标准**：

| 等级 | 初级 | 中级 | 高级 | 资深 |
|-----|-----|-----|-----|-----|
| A | 全面掌握 | 精通 | 专家指导 | 领域权威 |
| B | 熟练掌握 | 独立承担 | 带教他人 | 系统性输出 |
| C | 基本掌握 | 能独立完成 | 指导初级 | 创新贡献 |
| D | 了解 | 需指导 | 独立完成常规 | 指导他人 |
| E | 不了解 | 经常需要帮助 | 需要上级支持 | - |

### 3.3 各等级具体表现

**A（卓越）**：

- 显著超出岗位要求
- 工作成果质量极高
- 主动承担额外责任
- 对团队有显著贡献
- 具备晋升能力

**B（优秀）**：

- 超过岗位要求
- 工作成果质量高
- 主动完成本职工作
- 积极配合团队
- 持续发展潜力

**C（良好）**：

- 达到岗位要求
- 工作成果质量合格
- 完成本职工作
- 无消极表现
- 稳定贡献

**D（需改进）**：

- 部分未达岗位要求
- 工作质量有待提高
- 需上级督促
- 有改进意愿
- 需制定改进计划

**E（不达标）**：

- 明显未达岗位要求
- 工作失误较多
- 态度消极
- 无法独立工作
- 需要立即改进或调整

## 四、考核流程

### 4.1 月度考核

**时间安排**：

| 节点 | 时间 | 内容 |
|-----|------|-----|
| 提交小结 | 每月5日前 | 员工提交上月工作小结 |
| 上级评价 | 每月10日前 | 上级完成评价 |
| 结果反馈 | 每月15日前 | 绩效面谈和反馈 |

**月度小结内容**：

```
1. 本月主要工作完成情况
2. 工作成果和贡献
3. 遇到的问题和改进建议
4. 下月工作计划
```

### 4.2 季度考核

**时间安排**：

| 节点 | 时间 | 内容 |
|-----|------|-----|
| 自评 | 季度末月20日 | 员工自评 |
| 上级评价 | 季度末月25日 | 上级评分 |
| 校准会议 | 季度末月28日 | 部门绩效校准 |
| 结果反馈 | 下季度5日前 | 绩效面谈 |

**考核材料**：

| 材料 | 说明 |
|-----|------|
| 季度目标 | 季度初制定的目标 |
| 工作小结 | 月度小结汇总 |
| 自评表 | 员工自我评价 |
| 上级评价 | 直属上级评价 |

### 4.3 年度考核

**时间安排**：

| 节点 | 时间 | 内容 |
|-----|------|-----|
| 全年总结 | 12月15日 | 员工提交全年总结 |
| 年度评价 | 12月20日 | 上级完成评价 |
| 年终校准 | 12月25日 | 公司绩效校准 |
| 结果反馈 | 12月31日 | 年度绩效面谈 |

**年度考核重点**：

- 全年目标完成情况
- 核心能力发展
- 潜力评估
- 晋升资格认定
- 年度人才盘点

### 4.4 绩效校准

**校准目的**：

- 确保部门内评分公正
- 平衡不同员工的表现差异
- 与公司整体绩效挂钩

**校准流程**：

```
部门汇报 → 横向对比 → 调整评分 → 确定等级 → 汇总上报
```

**校准原则**：

- 同一部门内相似表现应有相似评分
- 等级分布应符合公司要求
- 特殊情况需说明理由

## 五、绩效面谈

### 5.1 面谈要求

**面谈时间**：

- 季度考核后一周内完成
- 年度考核后一周内完成
- 时间不少于30分钟

**面谈参与人**：

| 类型 | 参与人 |
|-----|-------|
| 普通员工 | 直属上级 + HRBP |
| 管理人员 | 直属上级 + 人事部 |
| 高潜人才 | 上级 + HR + 高管 |

### 5.2 面谈内容

**面谈话术模板**：

```
1. 开场（5分钟）
   - 营造轻松氛围
   - 说明面谈目的

2. 绩效回顾（10分钟）
   - 本周期工作表现
   - 主要成果和不足

3. 反馈讨论（10分钟）
   - 具体事例说明
   - 双向沟通

4. 发展规划（5分钟）
   - 下期目标设定
   - 能力提升计划

5. 总结（5分钟）
   - 确认改进计划
   - 提供支持资源
```

### 5.3 绩效改进计划（PIP）

**适用情况**：

- 连续两个季度绩效为D
- 或一个季度绩效为E
- 绩效明显低于岗位要求

**PIP流程**：

```
启动PIP → 制定计划 → 执行监督 → 评估结果 → 转正/调岗/离职
```

**PIP周期**：通常为60-90天

**PIP内容**：

| 内容 | 说明 |
|-----|------|
| 现状分析 | 差距和原因 |
| 改进目标 | 具体可衡量的目标 |
| 改进措施 | 具体的行动方案 |
| 评估标准 | 达标的判断标准 |
| 资源支持 | 培训、辅导等支持 |

## 六、结果应用

### 6.1 薪酬调整

**年度调薪**：

| 绩效等级 | 调薪幅度 | 说明 |
|---------|---------|-----|
| A | 15-25% | 重点激励 |
| B | 8-15% | 正常调薪 |
| C | 3-8% | 基础调薪 |
| D | 0-3% | 微量调薪或不调 |
| E | 0% | 不调薪 |

**特别调薪**：

- 绩效突出者可申请特别调薪
- 需部门负责人和HR共同审批

### 6.2 奖金发放

**年终奖系数**：

```
年终奖 = 月基本工资 × 月数 × 绩效系数 × 公司业绩系数
```

| 绩效等级 | 绩效系数 |
|---------|---------|
| A | 2.0-3.0 |
| B | 1.2-1.5 |
| C | 1.0 |
| D | 0.5 |
| E | 0 |

### 6.3 晋升应用

| 绩效要求 | 晋升条件 |
|---------|---------|
| 基本要求 | 最近1年平均B+ |
| 优先考虑 | 连续两年A或多个A |
| 否决条件 | 任何一年E |

### 6.4 培训发展

| 绩效等级 | 培训安排 |
|---------|---------|
| A | 优先安排外部培训、轮岗 |
| B | 部门内训、导师辅导 |
| C | 标准培训计划 |
| D | 专项能力提升培训 |
| E | PIP或岗位调整 |

## 七、常见问题

### 7.1 考核问题

**Q：对自己评分不满意怎么办？**
A：可以先与直属上级沟通，如仍不满意可向HRBP申诉。

**Q：绩效等级是怎么分布的？**
A：公司有强制分布比例，A≤10%，B20-30%，C40-50%，D10-15%，E≤5%。

**Q：缺勤会影响绩效考核吗？**
A：会，实际出勤率会影响最终评分，需在评分时说明。

### 7.2 反馈问题

**Q：绩效考核多久反馈一次？**
A：至少每季度一次正式反馈，月度有小结沟通。

**Q：如何获得上级更好的评价？**
A：主动沟通工作进展，争取高难度任务，及时反馈问题。

**Q：绩效差会被解雇吗？**
A：绩效差会先进入PIP改进，仍不达标可能会调整岗位或协商离职。

### 7.3 申诉问题

**Q：对考核结果有异议怎么办？**
A：在收到结果7天内向HRBP提出书面申诉。

**Q：申诉多久会有回复？**
A：HRBP会在7个工作日内处理并回复。

**Q：申诉会影响我的工作吗？**
A：不会，申诉是员工的正当权利，公司保护申诉人不受打击报复。
