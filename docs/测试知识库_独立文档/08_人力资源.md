# 人力资源制度

本文档详细介绍员工守则、离职流程、员工关系、劳动争议等人力资源相关内容。

---

## 一、员工守则

### 1.1 基本职业道德

**爱岗敬业**：

- 热爱本职工作，履行工作职责
- 遵守劳动纪律，完成工作任务
- 不断提高专业技能

**诚实守信**：

- 言行一致，信守承诺
- 不弄虚作假，不欺诈隐瞒
- 保守公司秘密

**勤勉尽责**：

- 工作认真负责，不推诿拖延
- 主动承担工作，主动改进
- 维护公司利益，不损害公司声誉

### 1.2 行为规范

**着装规范**：

| 场合 | 着装要求 | 禁止 |
|-----|---------|-----|
| 日常工作 | 商务休闲装 | 拖鞋、背心 |
| 正式会议 | 正装或商务装 | 牛仔裤、运动装 |
| 客户拜访 | 正装 | 休闲装 |
| 团建活动 | 休闲装 | 无要求 |

**工作态度**：

- 准时参加会议，提前5分钟到场
- 及时回复邮件和信息（24小时内）
- 积极参与团队活动
- 保持办公区域整洁

**沟通规范**：

- 使用文明礼貌用语
- 尊重同事和客户
- 保持积极正面的态度
- 建设性反馈意见

### 1.3 禁止行为

**严重违纪行为**（直接解除劳动合同）：

| 行为类型 | 具体表现 |
|---------|---------|
| 违反考勤 | 连续旷工3天或累计7天 |
| 违反制度 | 严重违反公司规章制度 |
| 损害利益 | 营私舞弊、损害公司利益 |
| 诚信问题 | 欺诈、弄虚作假 |
| 违法行为 | 被依法追究刑事责任 |
| 泄露秘密 | 泄露公司机密 |
| 打架斗殴 | 殴打他人或相互斗殴 |

**一般违纪行为**（视情况处理）：

| 行为类型 | 处理方式 |
|---------|---------|
| 迟到早退 | 扣款处理 |
| 消极怠工 | 警告+扣款 |
| 损坏公物 | 照价赔偿 |
| 旷工1-2天 | 扣款+警告 |
| 私用公司资源 | 警告 |

### 1.4 利益冲突

**需要申报的情况**：

| 类型 | 说明 | 申报时间 |
|-----|------|---------|
| 兼职 | 从事与公司业务相关的兼职 | 入职时及变更时 |
| 投资 | 投资与公司有业务往来的企业 | 入职时及变更时 |
| 亲属就业 | 亲属在公司或关联公司就业 | 入职时及变更时 |

**申报流程**：

1. 填写利益冲突申报表
2. 直属上级审核
3. 人事部备案
4. 必要时回避相关工作

## 二、离职流程

### 2.1 主动离职

**提前通知**：

| 员工类型 | 提前通知期 | 说明 |
|---------|----------|-----|
| 试用期员工 | 3天 | 书面通知 |
| 转正员工 | 30天 | 书面通知 |

**离职申请流程**：

```
员工提出书面申请 → 直属上级沟通 → 工作交接 → 资产归还 → 结算手续 → 离职
```

**详细步骤**：

**第一步：提交离职申请**（D-30或D-3）

1. 提交书面离职申请（邮件或纸质）
2. 说明离职原因和最后工作日
3. 直属上级收到申请后沟通

**第二步：工作交接**（D-30至D-0）

1. 整理工作文件和个人资料
2. 编写交接清单
3. 向接替者或上级详细交接
4. 更新待办事项清单
5. 告知合作方工作变更

**第三步：资产归还**（最后工作日）

1. 联系IT部门归还设备
2. 联系行政部归还门禁卡、工牌
3. 归还公司图书、设备

**第四步：结算手续**（最后工作日）

1. 结算最后一个月工资
2. 结算未休年假折算
3. 结算未报销费用
4. 办理社保和公积金转移

**第五步：离职手续**（最后工作日）

1. 签署离职协议
2. 提交离职满意度调查
3. 领取离职证明
4. 退出工作群和系统

### 2.2 被动离职（裁员）

**适用情况**：

| 情况 | 说明 |
|-----|------|
| 经济性裁员 | 公司经营困难需要裁员 |
| 岗位撤销 | 组织架构调整 |
| 绩效不达标 | 连续两年绩效E |
| 违反制度 | 严重违纪 |

**裁员流程**：

```
公司通知 → 协商解除 → 签署协议 → 工作交接 → 结算手续 → 离职
```

**补偿标准**：

| 在司年限 | 经济补偿 |
|---------|---------|
| 6个月以下 | 半个月工资 |
| 6个月-1年 | 1个月工资 |
| 1-3年 | N个月工资（N=年限） |
| 3-6年 | 每满一年1个月 |
| 6年以上 | 每满一年1.5个月 |

**工资计算**：

- 上年度月平均工资（12个月平均）
- 超过社会平均工资3倍按3倍计算
- 最长不超过12个月

### 2.3 离职交接

**交接清单**：

| 交接内容 | 负责人 | 接收人 |
|---------|-------|-------|
| 工作文件 | 本人 | 接替者或上级 |
| 项目进度 | 本人 | 接替者或上级 |
| 待办事项 | 本人 | 接替者或上级 |
| 账号权限 | 本人 | IT部门 |
| 门禁卡、工牌 | 本人 | 行政部 |
| 借款、预支 | 本人 | 财务部 |
| 公司资产 | 本人 | 行政部 |

**交接要求**：

- 交接内容必须完整
- 所有密码和账号必须更改
- 重要文件必须归档
- 交接双方和上级签字确认

### 2.4 离职结算

**工资结算**：

| 项目 | 说明 |
|-----|------|
| 最后一个月 | 按实际出勤计算 |
| 未休年假 | 折算3倍工资 |
| 绩效奖金 | 按实际完成情况发放 |
| 报销欠款 | 从工资扣除 |

**离职证明**：

- 离职后3个工作日内开具
- 内容：工作年限、岗位、离职时间
- 格式：统一模板

**社保公积金**：

| 操作 | 时间 | 说明 |
|-----|------|-----|
| 社保减员 | 离职当月 | 公司当月减员 |
| 公积金封存 | 离职当月 | 账户封存 |
| 转移办理 | 离职后 | 员工自行办理 |

## 三、员工关系

### 3.1 沟通渠道

**常规沟通**：

| 渠道 | 频率 | 内容 |
|-----|------|-----|
| 部门周会 | 每周 | 工作汇报和计划 |
| 月度例会 | 每月 | 部门总结和计划 |
| 季度座谈会 | 每季度 | 员工关系部组织 |
| 年度大会 | 每年 | 公司年度总结 |

**反馈渠道**：

| 渠道 | 说明 | 保密性 |
|-----|------|-------|
| 直属上级 | 日常沟通首选 | 不保密 |
| HRBP | 人事问题沟通 | 部分保密 |
| 匿名意见箱 | 意见反馈 | 完全保密 |
| 总经理邮箱 | 重大问题反馈 | 完全保密 |

### 3.2 员工关怀

**生病慰问**：

| 情况 | 关怀方式 |
|-----|---------|
| 门诊休息 | 部门同事代为慰问 |
| 住院 | HR或直属上级探望 |
| 长期病假 | 定期关怀，发放慰问品 |

**生活关怀**：

| 情况 | 关怀方式 |
|-----|---------|
| 结婚 | 发放1000元礼金 |
| 生育 | 发放500元礼金 |
| 生日 | 蛋糕券+生日会 |
| 退休 | 举办欢送会+纪念品 |

**困难员工**：

- 因病因灾致困可申请困难补助
- 补助金额根据情况确定
- 由员工关系委员会审批

### 3.3 员工活动

**活动类型**：

| 类型 | 频率 | 说明 |
|-----|------|-----|
| 部门团建 | 每月1次 | 人均100元/月 |
| 公司全员 | 每季度1次 | 公司统一组织 |
| 年度旅游 | 每年1次 | 人均2000元 |
| 节日活动 | 节日期间 | 春节、中秋等 |

**团建要求**：

- 自愿参加
- 不得酒后驾车
- 注意人身安全
- 费用需合规报销

## 四、劳动争议

### 4.1 争议预防

**预防措施**：

| 措施 | 说明 |
|-----|------|
| 明确合同 | 合同条款清晰明确 |
| 制度公示 | 规章制度员工知悉 |
| 及时沟通 | 问题早发现早解决 |
| 合规管理 | 依法依规管理 |

**沟通机制**：

1. 问题发生时先与直属上级沟通
2. 无法解决可找HRBP协调
3. 人事部调解
4. 调解不成申请仲裁

### 4.2 争议处理

**内部处理**：

| 阶段 | 处理方式 | 时效 |
|-----|---------|-----|
| 第一阶段 | 与直属上级沟通 | 7天内 |
| 第二阶段 | HRBP协调 | 7天内 |
| 第三阶段 | 人事部调解 | 10天内 |
| 第四阶段 | 劳动仲裁 | - |

**劳动仲裁**：

| 项目 | 说明 |
|-----|------|
| 受理机构 | 当地劳动争议仲裁委员会 |
| 申请时间 | 争议发生之日起1年内 |
| 所需材料 | 身份证、劳动合同、证据材料 |
| 仲裁费用 | 免费 |

### 4.3 常见争议类型

| 类型 | 常见原因 | 预防措施 |
|-----|---------|---------|
| 工资争议 | 加班费、扣款、拖欠 | 按时发薪、加班申请 |
| 社保争议 | 未缴、少缴 | 了解政策、及时缴纳 |
| 合同争议 | 变更、解除 | 合同明确、协商一致 |
| 离职争议 | 赔偿、交接 | 按规办理 |

## 五、劳动合同

### 5.1 合同类型

| 类型 | 适用情况 | 期限 |
|-----|---------|-----|
| 固定期限 | 一般员工 | 3年 |
| 无固定期限 | 连续两次签订后/10年以上 | 不固定 |

### 5.2 合同变更

| 变更类型 | 条件 | 流程 |
|---------|-----|-----|
| 岗位调整 | 协商一致 | 签订变更协议 |
| 薪资调整 | 协商一致 | 签订变更协议 |
| 合同续签 | 双方同意 | 重新签订 |

### 5.3 合同解除

| 解除类型 | 条件 | 赔偿 |
|---------|-----|-----|
| 协商解除 | 双方同意 | N个月工资 |
| 员工过失 | 严重违纪 | 无赔偿 |

## 六、职业健康

### 6.1 职业健康保护

**工作环境**：

- 符合国家职业健康标准
- 定期检测空气质量、噪音
- 提供必要的防护用品

**健康检查**：

| 类型 | 频率 | 说明 |
|-----|------|-----|
| 入职体检 | 入职时 | 必做 |
| 年度体检 | 每年 | 公司付费 |
| 特殊岗位体检 | 按需 | 接触职业危害岗位 |

### 6.2 职业病防护

**防护措施**：

- 接触职业危害岗位配备防护用品
- 定期职业健康培训
- 职业病危害告知

**疑似职业病**：

1. 及时就医
2. 申请职业病诊断
3. 公司配合提供资料
4. 依法享受相应待遇

## 七、常见问题

### 7.1 入职转正问题

**Q：试用期表现不好怎么办？**
A：公司会提前通知，提供改进机会。如仍不达标，可延长试用期或解除合同。

**Q：转正后工资什么时候调整？**
A：转正当月起按100%工资标准发放，调整在次月15日体现。

### 7.2 离职问题

**Q：离职工资什么时候结清？**
A：最后工作日结清所有工资，包括未休年假折算。

**Q：离职后社保怎么办？**
A：离职当月减员，新单位入职后续交。可自行办理转移。

**Q：离职需要赔偿培训费吗？**
A：签订过培训协议且服务期未满的，需要按比例退还培训费用。

### 7.3 劳动权益问题

**Q：加班费怎么算？**
A：工作日加班=日工资÷8×1.5×加班小时；周末加班=日工资÷8×2×加班小时。

**Q：年假没休完怎么办？**
A：因公司原因无法安排的，可折算3倍工资。公司安排休完的，当年休完。

**Q：生育津贴怎么领？**
A：产假期间由生育保险基金支付，员工无需操作，公司代为申领。
