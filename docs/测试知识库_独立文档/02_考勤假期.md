# 考勤与假期制度

本文档详细介绍公司考勤制度、请假流程、各类假期规定等内容。

---

## 一、工作时间

### 1.1 标准工时制

公司实行标准工时制，具体安排如下：

| 项目 | 时间 | 说明 |
|-----|------|-----|
| 上班时间 | 9:00 | 弹性范围9:00-9:30 |
| 下班时间 | 18:00 | 弹性范围17:30-18:30 |
| 午休时间 | 12:00-13:00 | 1小时，视为工作时间 |
| 每周工作日 | 周一至周五 | 周六周日为休息日 |
| 每天工时 | 8小时 | 午休1小时计入 |
| 每周工时 | 40小时 | 标准周工时 |

### 1.2 弹性工作时间

**申请条件**：

- 入职满3个月
- 绩效评分B及以上
- 特定岗位可申请（需部门负责人审批）

**弹性规则**：

- 上班时间可在9:00-9:30之间选择
- 下班时间相应调整为17:30-18:30
- 选择后需稳定执行至少3个月

**申请流程**：

1. 在OA系统提交弹性工作申请
2. 选择希望的上班时间（9:00或9:30）
3. 直属上级审批
4. HR备案
5. 次月1日起生效

### 1.3 特殊工时制

**不定时工时制**：

- 适用于高管、销售外勤、司机等岗位
- 不进行考勤打卡考核
- 按工作完成情况考核

**综合工时制**：

- 适用于项目制岗位
- 以月/季/年为周期计算工时
- 周期内总工时达到法定要求即可

## 二、考勤打卡

### 2.1 打卡方式

公司采用双重打卡方式：

**门禁卡打卡**：

- 进门时刷卡，系统自动记录
- 位置：1层大堂、3-6层电梯口

**企业微信定位打卡**：

- 上班前在公司范围内（半径200米）打开企业微信
- 点击"打卡"按钮
- 系统自动定位并记录时间

**打卡时间记录**：

| 打卡类型 | 记录时间 | 说明 |
|---------|---------|-----|
| 上班打卡 | 9:00-9:30 | 弹性范围内 |
| 下班打卡 | 17:30-18:30 | 弹性范围内 |
| 午休开始 | 12:00前 | 自动记录 |
| 午休结束 | 13:00后 | 自动记录 |

### 2.2 补卡申请

**补卡规则**：

- 每月有3次补卡机会
- 需说明补卡原因
- 直属上级审批后生效
- 第4次起需总监审批

**补卡申请流程**：

1. 登录OA系统
2. 进入"考勤管理"→"补卡申请"
3. 选择补卡日期和时间
4. 填写补卡原因（必填）
5. 提交审批

**补卡原因参考**：

- 忘记打卡（最常见）
- 门禁故障
- 定位失败
- 紧急外出无法打卡

**不予补卡的情况**：

- 外出办事但未提前报备
- 恶意不打卡
- 补卡理由虚假

### 2.3 外勤打卡

**适用场景**：

- 出差
- 客户拜访
- 外出办事

**外勤打卡流程**：

1. 出发前在OA系统提交"外出申请"
2. 选择外出地点（填写地址）
3. 在外出地点打开企业微信定位打卡
4. 返回后提交"外出归来"记录

## 三、迟到早退与旷工

### 3.1 迟到处理

**迟到认定**：

- 上班打卡时间晚于9:30（弹性结束后）
- 或企业微信定位不在公司范围

**迟到处理标准**：

| 迟到时长 | 处理方式 | 每月次数限制 |
|---------|---------|-------------|
| 15分钟以内 | 不扣款，但计入考勤 | 3次弹性 |
| 15-30分钟 | 扣款50元/次 | 无限制 |
| 30-60分钟 | 扣款100元/次 | 无限制 |
| 60分钟以上 | 扣款当日半天工资 | 无限制 |

**计算示例**：

- 月基本工资8000元，日工资=8000÷21.75=367.82元
- 迟到30分钟扣100元
- 迟到60分钟以上扣半天工资=183.91元

**特别说明**：

- 每月前3次15分钟内的迟到不扣款（视为弹性）
- 恶劣天气（红色/橙色预警）不计入迟到
- 地铁/公交故障（官方通报）需提供证明

### 3.2 早退处理

**早退认定**：

- 下班打卡时间早于17:30
- 或上班后擅自离开公司

**早退处理标准**：

| 早退时长 | 处理方式 |
|---------|---------|
| 30分钟以内 | 扣款50元/次 |
| 30-60分钟 | 扣款100元/次 |
| 60分钟以上 | 扣款当日半天工资 |

### 3.3 旷工处理

**旷工认定**：

- 未请假或未获批准不上班
- 未经批准擅自离开岗位超过2小时
- 虚假请假理由
- 补卡申请未通过且未出勤

**旷工处理标准**：

| 旷工时长 | 处理方式 |
|---------|---------|
| 1天 | 扣款3天工资 |
| 连续3天 | 视为严重违纪，解除劳动合同 |
| 累计7天 | 视为严重违纪，解除劳动合同 |

**旷工期间**：

- 不享受餐补（30元/天）
- 不享受交通补贴（当日）
- 不享受绩效工资

## 四、请假类型与规定

### 4.1 年假

**年假天数**：

| 工龄 | 年假天数 | 说明 |
|-----|---------|-----|
| 试用期 | 按转正天数折算 | 例：入职2个月，折算0.5天 |
| 满1年 | 5天 | 工作满1年后的首个周年起 |
| 满3年 | 10天 | 第三个周年起 |
| 满5年 | 15天 | 第五个周年起 |
| 满10年 | 20天 | 第十个周年起 |

**年假计算规则**：

- 以入职日期为基准计算周年
- 新入职员工：年假天数 =（剩余日历天数 ÷ 365）× 年应休天数
- 离职员工：未休年假按日工资3倍折算

**年假使用规则**：

1. 年假需在当年度内休完
2. 年假可拆分使用，最小单位为半天
3. 请假需提前3天申请
4. 紧急情况可先电话通知，事后补申请
5. 年假过期未休，不可跨年累计（可结转最多5天）

**年假申请流程**：

1. OA系统提交请假申请
2. 选择请假类型"年假"
3. 选择开始和结束日期
4. 系统自动计算请假天数
5. 提交直属上级审批
6. HR备案

### 4.2 病假

**病假申请条件**：

- 员工本人患病
- 需要治疗或休息

**病假天数与要求**：

| 病假天数 | 需要提供 | 工资发放标准 |
|---------|---------|------------|
| 3天以内 | 医院诊断证明 | 100% |
| 4-7天 | 诊断证明+病历 | 80% |
| 8-30天 | 完整病历+检查报告+诊断证明 | 60% |
| 30天以上 | 需公司医务室评估 | 停发基本工资 |

**病假申请流程**：

1. 提前1小时通过企业微信告知直属上级
2. OA系统提交病假申请
3. 上传医院证明（照片或扫描件）
4. 直属上级审批
5. 如需延长，提前1天续假

**疾病急诊**：

- 突发疾病可直接就医
- 24小时内由家属或本人补办请假手续
- 急诊需提供急诊证明

**长期病假**：

- 30天以上的病假需提供三甲医院证明
- 公司有权要求员工到指定医院复查
- 病假期间员工关系保留

### 4.3 事假

**事假规定**：

| 项目 | 规定 |
|-----|------|
| 全年累计上限 | 10个工作日 |
| 请假单位 | 最小0.5天 |
| 工资扣除 | 扣除当日全额工资（含绩效） |
| 餐补 | 不享受当日餐补 |
| 交通补贴 | 不享受当日交通补贴 |

**事假申请流程**：

1. 提前3天在OA系统提交
2. 填写请假原因（必须说明）
3. 直属上级审批
4. 紧急事假需电话通知，事后24小时内补申请

**不予批准的情况**：

- 理由不充分
- 部门工作繁忙且无人顶岗
- 频繁请事假（当月已请3天以上）

### 4.4 婚假

**法定婚假**：

| 类型 | 天数 | 说明 |
|-----|-----|-----|
| 基础婚假 | 3天 | 适用于所有员工 |
| 晚婚增加 | 7天 | 男满25周岁、女满23周岁 |

**申请条件**：

- 需在领取结婚证后1年内使用
- 必须一次性连续休完

**申请材料**：

- 结婚证复印件
- 休假申请表

**申请流程**：

1. 提前1周提交婚假申请
2. 上传结婚证
3. 直属上级审批
4. HR备案

### 4.5 产假

**产假天数**：

| 分娩情况 | 基础产假 | 奖励假 | 合计 |
|---------|---------|-------|-----|
| 顺产 | 98天 | 30天 | 128天 |
| 难产 | 98天 | 15天 | 143天 |
| 多胞胎 | 98天 | 每多1个+15天 | - |
| 流产 | 42天（怀孕满4个月） | - | - |
| 流产 | 15天（怀孕不满4个月） | - | - |

**申请材料**：

- 医院诊断证明
- 预产期证明
- 生育服务证（准生证）

**产假期间待遇**：

- 工资由生育保险基金支付生育津贴
- 公司不额外支付工资
- 生育津贴低于本人工资标准的，差额部分由公司补足

**产假申请流程**：

1. 产假前30天提交申请
2. 上传医院证明
3. 直属上级和HR共同审批
4. 产假期间社保正常缴纳

### 4.6 陪产假

**陪产假天数**：

| 类型 | 天数 | 说明 |
|-----|-----|-----|
| 基础陪产假 | 10天 | 适用于所有男员工 |
| 晚育增加 | 3天 | 夫妻双方晚婚晚育 |

**申请条件**：

- 妻子生育
- 在孩子出生后30天内使用

**申请材料**：

- 结婚证
- 孩子出生证明

### 4.7 丧假

**丧假天数**：

| 亲属关系 | 天数 | 说明 |
|---------|-----|------|
| 父母、配偶、子女 | 5天 | 含来回路程 |
| 祖父母、外祖父母 | 3天 | 含来回路程 |
| 岳父母、公婆 | 3天 | 含来回路程 |
| 兄弟姐妹 | 2天 | 含来回路程 |

**路程假**：

- 省外往返可增加1-2天路程假
- 需提供票据证明

**申请材料**：

- 死亡证明
- 亲属关系证明

## 五、加班管理

### 5.1 加班认定

**加班类型**：

| 类型 | 认定标准 | 说明 |
|-----|---------|-----|
| 工作日加班 | 下班后继续工作超过30分钟 | 需提前申请 |
| 周末加班 | 周六周日工作 | 需提前申请 |
| 节假日加班 | 国家法定节假日 | 需提前申请 |

**不视为加班的情况**：

- 工作效率低导致的工作延迟
- 个人原因未完成工作
- 出差途中时间
- 参加公司组织的娱乐活动

### 5.2 加班申请流程

**提前申请**：

1. 提前1天在OA系统提交加班申请
2. 填写加班原因、开始时间、预计时长
3. 直属上级审批
4. 加班当天实际打卡记录

**紧急加班**：

- 可先加班，事后24小时内补申请
- 需说明紧急原因

### 5.3 加班调休规则

**调休时长计算**：

| 加班类型 | 调休时长 | 使用期限 |
|---------|---------|---------|
| 工作日加班 | 1:1 | 季度内（90天内） |
| 周末加班 | 1:2 | 半年内（180天内） |
| 节假日加班 | 1:3 | 年底前（12月31日前） |

**调休申请**：

1. 在OA系统提交调休申请
2. 选择要使用的调休时长
3. 直属上级审批
4. 调休按半天或全天为单位

**调休过期**：

- 到期未使用的调休时长自动清零
- 不予折算工资

### 5.4 加班费

**无法调休时的加班费**：

- 工作日加班费 = 日工资 ÷ 8 × 1.5 × 加班小时数
- 周末加班费 = 日工资 ÷ 8 × 2 × 加班小时数
- 节假日加班费 = 日工资 ÷ 8 × 3 × 加班小时数

**加班费与调休二选一**：

- 优先安排调休
- 确实无法安排调休的，支付加班费

### 5.5 加班管控

**加班上限**：

| 项目 | 上限 | 说明 |
|-----|-----|-----|
| 单日加班 | 4小时 | 超过需总监审批 |
| 月度加班 | 36小时 | 超过需总监审批 |
| 季度加班 | 108小时 | 超过需VP审批 |

**强制休息**：

- 单日工作超过12小时，需休息24小时
- 连续3天加班超过4小时/天，需休息1天

## 六、请假流程汇总

### 6.1 请假申请通用流程

```
员工提交申请（OA系统）
    ↓
填写请假信息（类型、日期、原因）
    ↓
上传证明材料（如需）
    ↓
直属上级审批（24小时内处理）
    ↓
HR备案（系统自动）
    ↓
申请成功，开始休假
```

### 6.2 请假注意事项

**审批时效**：

- 直属上级需在24小时内审批
- 如未审批，系统自动通过
- 紧急情况可电话通知

**请假生效**：

- 3天以内请假：提前1天申请
- 3-7天请假：提前3天申请
- 7天以上请假：提前1周申请

**销假**：

- 请假到期后自动销假
- 如需延长，提前1天续假
- 续假流程同请假流程

## 七、假期计算示例

### 7.1 年假计算示例

**示例1：新员工年假**

- 入职日期：2024年3月15日
- 转正日期：2024年6月15日（满1年）
- 2024年可休年假 =（291天 ÷ 365天）× 5天 ≈ 4天

**示例2：满1年员工年假**

- 入职日期：2023年6月1日
- 2024年6月1日满1年
- 2024年应休年假 = 5天

### 7.2 试用期请假示例

- 试用期：2024年3月15日-6月14日
- 基本工资：10000元/月
- 试用期工资：8000元/月（80%）
- 请事假1天扣款 = 8000 ÷ 21.75 = 367.82元

## 八、常见问题

### 8.1 考勤问题

**Q：忘记打卡怎么办？**
A：每月3次补卡机会，在OA系统申请补卡，说明原因即可。

**Q：恶劣天气迟到怎么办？**
A：政府发布红色/橙色预警后，打卡时间弹性处理，不计入迟到。

**Q：出差怎么打卡？**
A：出差前提交出差申请，出差期间无需打卡，但需在企业微信填写工作日志。

### 8.2 假期问题

**Q：年假可以分开休吗？**
A：可以，最小单位为半天。但建议连续休，以便工作安排。

**Q：年假过期没休可以折算工资吗？**
A：不可以，年假过期作废。但因公司原因无法安排休假的，可申请折算。

**Q：病假期间工资怎么算？**
A：按病假天数和医院证明等级发放，通常为正常工资的60%-100%。

**Q：产假期间社保怎么办？**
A：社保正常缴纳，个人部分由公司代扣代缴。

### 8.3 加班问题

**Q：加班必须申请吗？**
A：是的，必须提前申请并获批。事后补申请需说明原因。

**Q：加班可以换成工资吗？**
A：优先安排调休，确实无法调休的，按加班费标准支付。

**Q：出差算加班吗？**
A：出差途中时间不算加班，但周末在客户处工作可申请加班。
