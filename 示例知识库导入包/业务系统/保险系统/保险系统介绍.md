# 互联网保险系统介绍

## 一、系统概述

互联网保险系统是公司面向保险业务全流程的数字化核心平台，支撑从产品投保、核保理赔、客户服务到数据分析的完整业务链路。系统对接渠道（App、小程序、代理人门户、第三方合作平台）、保险公司核心系统、支付清算平台、医疗健康服务平台等多个外部系统，实现保险业务的线上化、自动化和智能化。

**核心定位**：
- 连接用户与保险产品的数字化桥梁
- 打通投保、核保、理赔、保全全流程
- 实现保险业务与健康医疗服务的生态整合

## 三、核心子系统与功能说明

### 3.1 产品域

**功能**：保险产品配置与管理

| 模块 | 说明 |
| --- | --- |
| 产品管理 | 产品创建、费率配置、保障条款设置 |
| 险种配置 | 主险、附加险、责任组合配置 |
| 费率引擎 | 保费计算规则、浮动因子配置 |
| 产品上下架 | 产品发布、停售、版本管理 |

**典型使用场景**：
- 上线一款新的百万医疗险产品
- 调整某产品费率或保障范围
- 配置限时促销活动（如节日加息、赠险）

### 3.2 投保域

**功能**：用户投保全流程

| 模块 | 说明 |
| --- | --- |
| 投保引导 | 产品展示、保障方案选择、健康告知 |
| 智能核保 | 根据健康告知自动给出核保结论 |
| 支付环节 | 保费支付（微信、支付宝、银行卡） |
| 保单生成 | 电子保单生成与发送 |

**典型使用场景**：
- 用户在 App 选购医疗险，完成健康告知并支付
- 核保结论为"拒保"时，向用户展示拒保原因和建议

### 3.3 承保域

**功能**：保险合同的正式生效管理

| 模块 | 说明 |
| --- | --- |
| 保单管理 | 保单信息维护、状态变更（生效、退保、失效） |
| 续期管理 | 续期保费扣费、续保通知、保证续保处理 |
| 批改管理 | 保单信息变更（投保人、被保人、受益人等） |
| 犹豫期管理 | 犹豫期退保处理 |

### 3.4 理赔域

**功能**：保险理赔全流程处理

| 模块 | 说明 |
| --- | --- |
| 理赔报案 | 用户提交理赔申请（线上化） |
| 材料收集 | 医疗单据、诊断证明、发票等影像上传 |
| 理赔审核 | 审核材料完整性、理赔条件认定 |
| 理赔结案 | 理赔款支付、理赔结论通知 |
| 理赔查询 | 理赔进度查询、理赔历史记录 |

**典型使用场景**：
- 用户就医后，通过 App 提交理赔申请，上传医疗发票和诊断证明
- 理赔系统自动识别费用项目，计算理赔金额
- 审核通过后，理赔款直接打款至用户账户

### 3.5 保全域

**功能**：保单存续期的各项变更服务

| 模块 | 说明 |
| --- | --- |
| 信息变更 | 投保人、被保人、受益人信息变更 |
| 受益人变更 | 变更保单受益人 |
| 联系方式变更 | 手机号、邮箱、地址变更 |
| 犹豫期退保 | 签收保单后在犹豫期内申请退保 |

### 3.6 风控域

**功能**：保险业务全链路风险控制

| 模块 | 说明 |
| --- | --- |
| 反欺诈模型 | 识别理赔欺诈行为（如故意夸大损失、虚假理赔） |
| 核保风控 | 核保阶段的风险识别与拦截 |
| 行为风控 | 异常投保行为检测（如带病投保、短期内重复投保） |
| 黑名单管理 | 欺诈失信名单管理与查询 |

### 3.7 客服域

**功能**：用户服务与沟通

| 模块 | 说明 |
| --- | --- |
| 智能客服 | FAQ 问答、保单查询、报案指引 |
| 人工客服 | 复杂问题转人工处理 |
| 投诉管理 | 用户投诉登记、处理与跟踪 |
| 用户教育 | 保险知识普及、产品说明 |

### 3.8 财务域

**功能**：资金结算与财务报表

| 模块 | 说明 |
| --- | --- |
| 保费结算 | 与保险公司、分销渠道的资金清算 |
| 理赔款支付 | 理赔款出款与对账 |
| 手续费结算 | 代理人/渠道佣金的计算与发放 |
| 财务报表 | 业务收支日报、月报、年报 |

## 四、典型业务流程

### 流程一：用户投保

```
用户浏览产品 → 健康告知 → 智能核保 → 支付保费 → 保单生效 → 电子保单发送
```

### 流程二：理赔申请

```
用户报案 → 材料上传 → 材料审核 → 理赔核定 → 理赔支付 → 结案通知
```

### 流程三：续期续保

```
续期提醒通知 → 保费自动扣款 → 续保成功/失败通知 → 保单状态更新
```

## 五、常见问题

**Q1：用户投保后，保单多久生效？**

A：一般支付成功后的次日零时生效（具体以产品条款为准）。电子保单会在支付成功后 24 小时内发送至用户注册邮箱。

**Q2：理赔申请需要准备哪些材料？**

A：常规理赔材料包括：理赔申请书、身份证件、诊断证明、医疗费用发票、出院小结。若为意外险，还需提供意外事故证明。具体材料清单可在 App 理赔申请页面查看。

**Q3：保证续保的产品，第二年会自动扣费吗？**

A：保证续保产品在到期前会发送续保通知，投保人需确认续保并完成保费支付。部分产品支持自动续期扣费，具体以产品规则为准。

**Q4：如何查询理赔进度？**

A：用户可通过 App → "我的保单" → 选择对应保单 → "理赔记录" 查看理赔进度，或联系客服咨询。
