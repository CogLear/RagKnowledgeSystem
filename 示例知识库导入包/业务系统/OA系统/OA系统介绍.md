# OA 系统介绍

## 一、系统概述

OA（Office Automation，办公自动化）系统是企业内部核心的协同管理平台，旨在将日常行政管理、业务审批和协作流程电子化、规范化，提升组织运转效率，降低沟通成本，实现"让数据多跑路、让员工少跑腿"。

**核心价值**：
- 审批流程线上化，告别纸质签字
- 信息透明可追溯，告别"不知道找谁"
- 协作数据沉淀，告别"每次重新来"

## 二、主要功能模块

### 2.1 审批中心（流程审批）

OA 系统的核心模块，覆盖企业日常行政和业务审批场景：

| 审批类型 | 说明 |
| --- | --- |
| 请假审批 | 年假、病假、事假、婚假、产假等假期申请与审批 |
| 差旅审批 | 出差申请、交通住宿预订、差旅报销 |
| 费用审批 | 日常报销、业务招待费、项目支出等 |
| 合同审批 | 合同用印、合同审批流 |
| 采购审批 | 办公采购、项目采购申请与审批 |
| 离职审批 | 员工离职申请、工作交接、权限回收 |

**核心优势**：
- 审批流程可视化，可实时查看当前节点和审批人
- 移动端随时审批，不受时间和地点限制
- 审批历史完整记录，可追溯可审计

### 2.2 考勤管理

| 功能 | 说明 |
| --- | --- |
| 打卡方式 | 支持门禁刷卡、手机定位、Wi-Fi 等多种方式 |
| 考勤统计 | 自动统计出勤、迟到、早退、请假等数据 |
| 异常处理 | 支持补卡申请、请假申请、节假日调休 |
| 考勤报表 | 支持月度/年度考勤汇总导出 |

### 2.3 公告与通知

- **全员公告**：公司重要政策、通知的发布渠道
- **部门通知**：部门内部信息传达
- **待办提醒**：审批超时、工作任务等提醒
- **消息中心**：统一查看所有系统通知

### 2.4 文档中心

- 企业制度文档查阅
- 常用表格模板下载
- 知识库文档沉淀
- 文档版本管理

### 2.5 会议管理

- 会议室预约与管理
- 会议通知与日程同步
- 视频会议集成
- 会议纪要归档

### 2.6 其他功能

- **任务协作**：项目任务分解、指派、进度跟踪
- **问卷调查**：内部意见收集、数据统计
- **CRM 集成**：部分 OA 系统与客户管理系统打通，支持销售流程审批

## 三、典型使用场景

### 场景一：新员工入职

1. HR 在 OA 系统创建入职流程
2. 员工收到账号开通通知（邮箱、企业微信/IM、OA）
3. 员工在 OA 系统完成入职培训确认、制度阅读确认
4. 考勤数据自动同步至工资核算

### 场景二：请假审批

1. 员工在 OA 系统「请假申请」发起请假
2. 选择请假类型、填写起止时间、上传证明（如需要）
3. 系统自动流转至直属经理审批
4. 审批通过后，考勤数据自动更新
5. 员工可在「我的请假」查看剩余年假余额

### 场景三：差旅申请与报销

1. 员工在 OA 系统提交「出差申请」
2. 填写出差目的、地点、时间、预算
3. 审批通过后，可预订平台对接的机票酒店（如有）
4. 出差返回后，在 OA 系统提交「差旅报销」
5. 关联出差申请单，上传发票，财务审核后打款

### 场景四：跨部门协作

1. 发起「工作联络单」或「协助申请」
2. 说明协作需求、期望完成时间
3. 对方部门负责人审批确认
4. 任务执行与进度同步
5. 完成后双向评价

## 五、常见问题

**Q1：审批被驳回了怎么办？**

A：在「我的申请」中找到被驳回的申请，查看审批意见，根据意见修改后重新提交。若有异议，可直接联系审批人沟通。

**Q2：休假申请已经审批通过，但临时想更改日期？**

A：在「我的申请」中找到已通过的请假记录，发起「变更申请」或取消后重新提交新的请假申请。注意：已过的假期无法撤销。

**Q3：找不到对应的审批类型怎么办？**

A：可在 OA 系统搜索栏输入关键词（如"报销"、"采购"）搜索对应流程。若确认公司未开通该流程，建议联系部门助理或 HR 确认。

**Q4：提交审批时选错了审批人？**

A：若审批尚未开始，可自行撤回后重新提交。若已进入审批流程，需联系当前审批人驳回后重新提交。
