# 差旅报销指南

## 一、差旅报销概述

差旅报销是员工因公外出（如客户拜访、项目实施、市场调研、参加会议等）产生的交通、住宿、餐饮等费用的报销流程。本文档说明差旅报销的申请流程、所需材料及注意事项。

---

## 二、差旅报销标准

### 2.1 交通费用

| 出行方式 | 标准说明 |
| --- | --- |
| 飞机 | 经济舱（国内）；因公特殊需要经审批后可乘坐公务舱 |
| 火车 | 高铁/动车二等座；长途（6小时以上）经审批可乘坐一等座 |
| 汽车 | 城市间大巴、网约车（需提供行程单据） |
| 市内交通 | 出租车费、地铁费、公交费（需提供发票） |

### 2.2 住宿费用

- **一线城市（北京、上海、深圳等）**：上限 600 元/晚
- **二线城市**：上限 400 元/晚
- **三线及以下城市**：上限 300 元/晚
- 住宿超标部分需自行承担，或提前申请特批

### 2.3 餐饮费用

- **补贴标准**：一线城市 150 元/天；二线城市 120 元/天；三线及以下 100 元/天
- 业务招待餐费需单独报销，不计入差旅餐饮补贴

## 三、差旅报销所需材料

### 3.1 必备材料清单

1. **出差申请单**（OA 系统审批通过）
   - 需包含：出差目的、地点、时间、预计费用
   - 审批人签字

2. **交通票据**
   - 机票/火车票/汽车票原件
   - 网约车行程单截图+发票
   - 过路过桥费发票（如有）

3. **住宿发票**
   - 增值税普通发票或专用发票
   - 发票抬头须为公司名称
   - 入住日期、离店日期、房费明细清晰

4. **餐饮发票**（如选择实报实销而非补贴）
   - 餐饮发票（增值税发票）
   - 需与出差时间、地点匹配

5. **差旅报告/总结**（部分公司要求）
   - 说明出差成果、完成情况

### 3.2 特殊情况材料

| 场景 | 额外材料 |
| --- | --- |
| 因公垫付机票改签 | 原机票+改签单+差价发票 |
| 客户招待餐费 | 餐饮发票+客户名单+招待事由说明 |
| 多人拼车 | 行程单+分摊说明+主办人发票 |
| 境外出差 | 护照出入境记录+外汇兑换水单 |

## 四、差旅报销流程

### 4.1 事前：出差申请

1. 在 OA 系统中提交「出差申请」
2. 填写：出差目的、目的地、预计出发/返回日期、预计费用
3. 选择报销方式（实报实销 / 餐饮补贴）
4. 提交直属经理审批

### 4.2 事中：费用保留凭证

- 所有费用务必索取发票
- 发票抬头必须与公司名称一致
- 发票日期需与出差行程匹配
- 建议出差期间每日记录花销，避免遗漏

### 4.3 事后：报销提交

1. 出差返回后 **5 个工作日内** 在 OA 系统提交报销
2. 上传所有发票影像/电子发票
3. 关联出差申请单
4. 提交财务审核
5. 审核通过后，报销款将打入工资卡

## 五、常见问题 FAQ

**Q1：发票不小心弄丢了怎么办？**

A：联系出票方补开发票（如航空公司官网可重新打印行程单）；若无法补开，需提供加盖公章的发票丢失证明，并附上其他付款凭证（如银行流水），经财务审批后可特殊处理。

**Q2：出差期间客户请客，餐费可以报销吗？**

A：可以，但需走「业务招待费」单独报销，不计入差旅餐饮补贴。需要提供：餐饮发票、客户名单（脱敏）、招待事由说明，按公司业务招待标准执行。

**Q3：住宿超标了可以报销吗？**

A：一般情况下超标部分不予报销。如因特殊情况（如当地无合规酒店、重大会议期间房价上涨）导致超标，需在出差申请中说明并提前获得特批，报销时附上特批记录。

**Q4：差旅补贴和实报实销可以混合使用吗？**

A：不可以。报销时需选择一种方式：要么选择餐饮补贴（不报销餐饮发票），要么选择实报实销（凭票报销餐饮费用）。

**Q5：出差期间加班的交通费可以报销吗？**

A：因公出差期间的加班交通费（如晚上返回酒店）可凭发票报销；但若为个人原因产生的交通费不在报销范围内。

## 六、注意事项

1. **时效要求**：出差返回后 **5 个工作日内** 提交报销，逾期可能影响报销或需额外说明
2. **发票合规**：必须是正规增值税发票，收据、白条不予报销
3. **行程一致**：发票日期和地点需与出差申请中的行程一致
4. **事前审批**：无事先审批的差旅费用，财务有权拒绝报销
5. **超标处理**：超标准费用需有特批记录方可报销
